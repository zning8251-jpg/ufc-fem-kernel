## Shared 域级合同卡 (L5_RT)

- **层级**: L5_RT  
- **域名**: Shared / 跨域共享类型与工具  
- **缩写**: Shared (`RT_Shared_*`、`RT_Shared_Type`)  
- **职责**: 定义**L5_RT层跨域共享**的TYPE定义、公共接口、工具函数，避免重复定义。**不对应** HYPLAS(共享逻辑分散)。  
- **非职责**: 不执行具体业务逻辑;不管理域特定状态。  

- **与 L3/L4 关系**:  
  - L3 `MD_Shared_*`: 基础TYPE → L5 `RT_Shared_Type` 扩展  
  - L4 `PH_Shared_*`: 物理层共享 → L5 引用或扩展  
  - **热路径零L3**: 共享TYPE仅作为数据结构，不包含L3调用。  

- **四型配置**（典型）：  
  - **Desc**: 共享TYPE定义(状态/配置/统计)。  
  - **State**: 跨域状态快照、全局计数器。  
  - **Ctx**: 共享上下文句柄、内存池引用。  
  - **Algo**: 无(纯数据定义域)。  

- **核心接口**（按功能集）：  

| 功能集 | 绑定 | 说明 |
|--------|------|------|
| Shared Types | `RT_Shared_Type` | 跨域共享TYPE定义;状态/配置/统计 |

- **依赖**: L1 `IF_Prec_Core` (精度定义)、L1 `IF_Err_*` (错误类型)。  
- **下游**: 所有L5_RT域 (Assembly/Solver/StepDriver等)。  
- **热路径**: **否** — 纯TYPE定义，运行时零开销。
