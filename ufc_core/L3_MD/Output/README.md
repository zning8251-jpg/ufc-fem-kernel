# Output_Field

Field output (spatial distribution) schema and export.

**Modules**:
- `MD_Out_Field_Export`: Read-only export (U, S, PEEQ, etc.) from Model for Output
- `MD_UniFld_Core`: Unified field type system and field manager
- `MD_UniFld_Ops`: Unified field operations
- `MD_Out_ReportPlot`: Report/Plot/PostProcessing/Animation/Export

**Reference**: ABAQUS Field Output, UFC Output Architecture.
