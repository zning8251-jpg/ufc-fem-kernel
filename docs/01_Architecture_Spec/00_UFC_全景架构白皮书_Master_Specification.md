## 第一章：核心愿景与架构哲学

### 1.1 终极愿景 (UniFieldCore)

UFC 的本质不再是一个传统的“固体力学计算器”，而是一个**场无关（Field-Agnostic）的大型偏微分方程（PDE）装配与求解引擎**。通过统一的数据底座和算子注入机制，UFC 能够以极高的扩展性同时覆盖固体大变形、高雷诺数流体（CFD）、高频电磁场（EM）等复杂多物理场计算。

### 1.2 极简三段式与六层架构

所有代码严格遵守 `[Prefix]_[Domain]_[Suffix]` 命名法。

- **六大层级**：`L1_BS` (基础工具) $\to$ `L2_NM` (数值数学) $\to$ `L3_MD` (物理模型) $\to$ `L4_PH` (网格与单元) $\to$ `L5_RT` (求解运行) $\to$ `L6_AP` (应用装配)。
- **后缀铁律**：四大类型（Desc, Ctx, State, Algo）仅作为 TYPE 结构体名，**严禁作为 `.f90` 文件后缀**。物理文件后缀收口为：`_Def` (数据定义), `_Core` (无状态算法), `_Proc` (流程管线), `_Brg` (外部桥接)。

## 第二章：数据底座与系统全生命周期

### 2.1 场注册表与混合自由度拓扑

- **废弃硬编码**：全局内存 `g_ufc_global` 彻底废除写死的 $(u,v,w)$ 位移概念。
- **场声明驱动**：应用层初始化时通过 `FieldManager` 动态注册（如 `RegisterField('Velocity', VECTOR, Order=2)`），支持泰勒-胡德等混合元和边单元，原生支持 LBB 鞍点问题。

### 2.2 扁平化数据真源（L3_MD）

- **打破物理嵌套**：废弃 ABAQUS 逻辑上 6 层深的物理嵌套。所有实体/材料/截面统一存入平行内存池，通过**外键 ID 关联**。物理 TYPE 嵌套深度被死死限制在 3 层以内，杜绝缓存穿透。

### 2.3 结构化 IO (SIO) 与软缓存

- **5/6 参铁律**：核心子程序签名强制为 `(desc, state, algo, ctx, [com_ctx], args)`。
- **软缓存机制**：`Ctx` 和 `Args` 在 `Populate` 时从庞大内存池中组装，作为不到 1KB 的“软件定义 L1 Cache 块”喂给 `_Core` 引擎，实现无指针乱飞的光速计算。

## 第三章：时空动三维正交与执行引擎

彻底消灭传统代码中“时间循环”与“空间组装”的维度坍缩歧义。执行字典采用 **3-LAR (三字母压缩法则)**，确保千万行代码如臂使指。

- **X 轴 (Where, 空间)**：`Glb` (全局), `Reg` (区域), `Elm` (单元), `Pt` (积分点), `**Fac` (面域，专为 DG-FEM 设计)**。
- **Y 轴 (When, 时相)**：宏观（`Stp`, `Inc`）$\times$ 微观（`Ini`, `Prd`, `Itr`, `Cmt`, `Rbk`）。
- **Z 轴 (What, 动作)**：`Pop` (捞取数据), `Evl` (核心测算), `Asm` (组装到 CSR), `Map` (场平滑映射), `Exp` (落盘输出)。
- **签名示例**：`_Proc_ElmItrEvl` (单元级迭代中测算)。

## 第四章：极限非线性与多场防御体系

为支撑强非线性和复杂流场，基座标配以下工业级防线：

1. **自动微分（AD）混合引擎**：经典本构保持原样；新材料基于双数（Dual Numbers）算符重载，自动生成零误差切线刚度 $D_{ep}$。
2. **算子注入与稳定化（VMS/SUPG）**：底层单元只算标准 Galerkin，组装前通过挂载点动态注入 SUPG / GLS 稳定化算子，根治对流占优震荡。
3. **大变形防暴走控制**：L5 标配**智能线搜索（Line Search）**防过冲，L4 标配 **B-bar 映射**解绑不可压体积锁死。
4. **接触平滑与热切换**：放弃生硬罚函数，引入**增广拉格朗日乘子更新环**；自带矩阵冷热探针，自动从对称静态求解器“热切换”至非对称 / 动态求解器跨越失稳点。

## 第五章：求解器矩阵与外挂数学库战略

解方程的数学脏活全部通过 `L2_NM_Brg` 隔离外包，UFC 专注物理与组装。

- **直接法防线**：中小规模与病态问题，接驳 Intel MKL PARDISO / MUMPS。
- **亿级扩展防线**：超大规模正定问题，接驳 PETSc + Hypre(BoomerAMG 代数多重网格) 预处理。
- **多场算子分裂管线（Fractional Step）**：针对流体巨大鞍点，L5_RT 状态机将方程拆散。动量步交由非对称 Krylov 迭代，压力泊松步交由绝对对称的 AMG 秒杀，从根本上规避矩阵奇异发散。
