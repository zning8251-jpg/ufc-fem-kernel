## 2. 多物理场广义自由度映射体系 (Generalized DOFs Mapping)

### 2.1 物理变量的抽象替换
在多物理场（固体力学、热传导、渗流、电磁）中，如果每个物理场都单独写一套求解器，代码将急剧膨胀且无法耦合。
**UFC 的核心策略：在物理域(L4)保留物理意义，在解算域(L5)统一替换为广义未知量 $U$。**

*   **广义位移 $U$**：代表系统所有的未知量。
*   **广义速度 $\dot{U}$ / 广义加速度 $\ddot{U}$**：分别对应温度率/孔压率（一阶），以及物理速度/加速度（二阶）。

### 2.2 DOF 槽位映射规则 (对标 ABAQUS 工业规范)
UFC 内核定义标准的 DOF 索引字典。L5 求解器只看到数字 1~30，只有 L4 物理域知道它们的具体含义：
*   **DOF 1-3**: $x, y, z$ 平移位移 ($u$)
*   **DOF 4-6**: $x, y, z$ 旋转 ($rot$)
*   **DOF 8**: 孔隙水压力 ($pore\_pressure$)
*   **DOF 11**: 温度 ($T$)

## 3. 统一瞬态时间积分方程 (Unified Transient Framework)

所有的工程问题最终都可归结为以下**统一的半离散系统微分方程**：
$$ M \ddot{U} + C \dot{U} + F^{int}(U) = F^{ext}(t) $$

### 3.1 降维与场景对应
通过控制质量矩阵 $M$ 和阻尼/传导矩阵 $C$，上述通用数学格式可以瞬间切换物理场景：
1.  **稳态/静力学 (Static)**: $M = 0, C = 0 \implies F^{int}(U) = F^{ext}$
2.  **一阶瞬态方程 (1st-Order Transient: 热传导、渗流)**:
    $M = 0 \implies C \dot{U} + F^{int}(U) = F^{ext}$ （此时 $C$ 为比热容或渗流储水矩阵）
3.  **二阶瞬态方程 (2nd-Order Transient: 结构动力学)**:
    $M \neq 0, C \neq 0$

### 3.2 增量格式与切线刚度统一
采用统一的时间积分算法（如 Generalized-$\alpha$ 或 Newmark-$\beta$），并运用牛顿-拉夫逊方法进行线性化，得到统一的增量求解格式：
$$ \hat{K} \Delta U = \hat{R} $$
其中：
*   **有效刚度**: $\hat{K} = c_1 M + c_2 C + K_T$ （$K_T = \frac{\partial F^{int}}{\partial U}$ 为材料与几何切线刚度，$c_1, c_2$ 为积分步长参数）
*   **有效残差**: $\hat{R} = F^{ext} - F^{int} - M \ddot{U} - C \dot{U}$

**架构映射：**
L4 物理层负责分别输出 $M_e, C_e, (K_T)_e$。
L5 解算层负责提供 $c_1, c_2$ 标量，并按照统一的数学公式组装有效刚度 $\hat{K}$，此时求解器完全不在乎它解的是热流还是固体。

## 7. 核心骨架：多场统一方程映射到代码的“标准流水线”

```fortran
SUBROUTINE PH_Elem_Unified_Transient_Eval(desc, state, algo, ctx, args)
    ! =====================================================================
    ! 【数学映射骨架: 多物理场统一下限的有效刚度与残差求值】
    ! 涵盖: 固体、热传导、渗流 (通过传入不同的 algo 参数选择性计算 M, C)
    ! =====================================================================
    
    ! [数学: 初始化积分结果] M_e=0, C_e=0, K_e=0, R_e=0
    ctx%local_mass      = 0.0d0
    ctx%local_damping   = 0.0d0
    ctx%local_stiffness = 0.0d0
    ctx%local_residual  = 0.0d0
    
    DO i_gp = 1, desc%num_gauss_points
        ! 1. 计算形函数与雅可比
        CALL PH_Shape_Eval(desc%coords, gp(i_gp), ctx%dN_dx, ctx%detJ)
        CALL PH_Kinematics_Build_B(ctx%dN_dx, ctx%B_Matrix)
        
        ! 2. 状态增量与内力/通量计算 (调用具体物理本构)
        CALL MD_Mat_Core_Eval(mat_desc, mat_state, mat_algo, mat_ctx)
        
        dV = ctx%detJ * desc%gauss_weights(i_gp)
        
        ! 3. [统一数学: M] 质量/电容矩阵累加 (二阶动力学必需)
        IF (algo%requires_mass) THEN
            ctx%local_mass = ctx%local_mass + mat_ctx%density * MATMUL(ctx%N, TRANSPOSE(ctx%N)) * dV
        END IF
        
        ! 4. [统一数学: C] 阻尼/热容/孔隙渗流储水矩阵累加 (一阶/二阶瞬态必需)
        IF (algo%requires_damping) THEN
            ctx%local_damping = ctx%local_damping + mat_ctx%capacity * MATMUL(ctx%N, TRANSPOSE(ctx%N)) * dV
        END IF
        
        ! 5. [统一数学: K_T] 物理本构切线刚度累加
        ctx%local_stiffness = ctx%local_stiffness + MATMUL(TRANSPOSE(ctx%B_Matrix), MATMUL(mat_ctx%Jacobian, ctx%B_Matrix)) * dV
            
        ! 6. [统一数学: F_int] 内力/内部通量残差累加
        ctx%local_residual = ctx%local_residual + MATMUL(TRANSPOSE(ctx%B_Matrix), mat_state%stress_or_flux) * dV
            
    END DO
END SUBROUTINE
```
