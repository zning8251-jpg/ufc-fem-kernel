## 1. L2_NM 层概述

### 1.1 层级定位

**L2_NM = 数值方法层（Numerical Methods Layer）**

**核心职责**:

- 提供纯数学算法（对矩阵物理含义透明）
- 线性/非线性求解
- 特征值求解
- 时间积分
- 稀疏矩阵存储与运算

**关键约束**:

- L2 不知道"刚度矩阵 K"代表什么，只知道"求解 A·x = b"
- FEM 语义属于 L4/L5，L2 只提供数值算法

**依赖约束**:

- ✅ 允许 USE: L1_IF
- ❌ 禁止 USE: L3-L6（通过 Bridge 域引入外部库）

### 1.2 三条铁律


| 铁律         | 内容                                 |
| ---------- | ---------------------------------- |
| **物理含义透明** | L2 所有算法对矩阵物理含义一无所知，可复用于任意 PDE 离散问题 |
| **外部库隔离**  | 外部库（MUMPS/cuSPARSE）只通过 Bridge 域引入  |
| **命名规范**   | UF_* 遗留文件须逐步迁移为 NM_* 前缀            |

## 2. 线性求解器理论

### 2.1 直接法（Direct Methods）

#### LU 分解

**分解**:

```
A = L U
```

其中:

- `L`: 下三角矩阵
- `U`: 上三角矩阵

**求解**:

```
Ly = b  (前向替换)
Ux = y  (后向替换)
```

**计算复杂度**: O(n³)（分解），O(n²)（求解）

#### LDLᵀ 分解（对称矩阵）

**分解**:

```
A = L D Lᵀ
```

其中:

- `L`: 下三角矩阵（单位对角）
- `D`: 对角矩阵

**求解**:

```
Ly = b
Dz = y
Lᵀx = z
```

**优势**: 对称矩阵只需存储一半

#### Cholesky 分解（正定对称矩阵）

**分解**:

```
A = L Lᵀ
```

其中 `L` 是下三角矩阵。

**条件**: `A` 必须正定对称

**优势**: 比 LDLᵀ 更快（约 2 倍）

### 2.2 迭代法（Iterative Methods）

#### 共轭梯度法（CG）

**算法**:

```
x₀ = 0
r₀ = b - Ax₀
p₀ = r₀

for k = 0, 1, 2, ...:
  αₖ = (rₖᵀrₖ) / (pₖᵀApₖ)
  xₖ₊₁ = xₖ + αₖpₖ
  rₖ₊₁ = rₖ - αₖApₖ
  βₖ₊₁ = (rₖ₊₁ᵀrₖ₊₁) / (rₖᵀrₖ)
  pₖ₊₁ = rₖ₊₁ + βₖ₊₁pₖ
```

**收敛条件**: `A` 必须对称正定

**收敛率**:

```
||eₖ||_A ≤ 2(√κ - 1)/(√κ + 1)ᵏ ||e₀||_A
```

其中 `κ = λ_max/λ_min` 是条件数。

#### GMRES（广义最小残差法）

**算法**:

```
x₀ = 0
r₀ = b - Ax₀
v₁ = r₀ / ||r₀||

for j = 1, 2, ..., m:
  w = Avⱼ
  for i = 1, ..., j:
    hᵢⱼ = vᵢᵀw
    w = w - hᵢⱼvᵢ
  hⱼ₊₁ⱼ = ||w||
  vⱼ₊₁ = w / hⱼ₊₁ⱼ
  
  Solve: y = argmin ||βe₁ - Hⱼy||
  xⱼ = Vⱼy
```

**优势**: 适用于非对称矩阵

**重启策略**: 每 `m` 步重启（防止内存增长）

#### BiCGStab（双共轭梯度稳定法）

**算法**:

```
x₀ = 0
r₀ = b - Ax₀
r̃₀ = r₀
p₀ = 0

for k = 0, 1, 2, ...:
  ρₖ = r̃₀ᵀrₖ
  if ρₖ == 0: break
  if k > 0:
    βₖ = (ρₖ/ρₖ₋₁)(αₖ₋₁/ωₖ₋₁)
    pₖ = rₖ + βₖ(pₖ₋₁ - ωₖ₋₁vₖ₋₁)
  else:
    pₖ = rₖ
  vₖ = Apₖ
  αₖ = ρₖ / (r̃₀ᵀvₖ)
  s = rₖ - αₖvₖ
  t = As
  ωₖ = (tᵀs) / (tᵀt)
  xₖ₊₁ = xₖ + αₖpₖ + ωₖs
  rₖ₊₁ = s - ωₖt
```

**优势**: 适用于非对称矩阵，无需重启

### 2.3 预条件（Preconditioning）

**预条件系统**:

```
M⁻¹Ax = M⁻¹b
```

其中 `M` 是预条件矩阵。

**常用预条件**:

- **对角预条件**: `M = diag(A)`
- **不完全 LU (ILU)**: `M ≈ LU`
- **不完全 Cholesky (IC)**: `M ≈ LLᵀ`
- **多重网格**: `M = 多重网格算子`

## 3. 非线性求解器理论

### 3.1 Newton-Raphson 方法

**非线性系统**:

```
R(u) = 0
```

**Newton-Raphson 迭代**:

```
Kₜ(uₖ) Δuₖ = -R(uₖ)
uₖ₊₁ = uₖ + Δuₖ
```

其中:

- `Kₜ = ∂R/∂u`: 切线刚度矩阵
- `R`: 残差向量

**收敛准则**:

```
||R(uₖ₊₁)|| < εᵣ ||R(u₀)||  (残差准则)
||Δuₖ₊₁|| < εᵤ ||uₖ₊₁||     (位移准则)
```

### 3.2 修正 Newton-Raphson

**算法**:

```
Kₜ(u₀) Δuₖ = -R(uₖ)  (Kₜ 不更新)
```

**优势**: 减少矩阵分解次数

**劣势**: 收敛速度降低（线性收敛）

### 3.3 弧长法（Riks Method）

**扩展系统**:

```
R(u, λ) = 0
```

其中 `λ` 是载荷因子。

**约束方程**:

```
ΔuᵀΔu + Δλ² = Δl²
```

**迭代方程**:

```
[Kₜ  -F₀] [Δu]   [-R]
[2Δuᵀ  2Δλ] [Δλ] = [0]
```

**优势**: 可追踪载荷-位移曲线的下降段

## 4. 特征值求解理论

### 4.1 Lanczos 方法

**算法**:

```
v₁ = random vector, ||v₁|| = 1
β₁ = 0
v₀ = 0

for j = 1, 2, ..., m:
  w = Avⱼ - βⱼvⱼ₋₁
  αⱼ = vⱼᵀw
  w = w - αⱼvⱼ
  βⱼ₊₁ = ||w||
  vⱼ₊₁ = w / βⱼ₊₁
  
  Tⱼ = tridiag(α, β)  (三对角矩阵)
  Solve: Tⱼy = θy  (特征值问题)
```

**优势**: 适用于大规模稀疏矩阵

**应用**: 结构动力学（模态分析）

### 4.2 Arnoldi 方法

**算法**:

```
v₁ = random vector, ||v₁|| = 1

for j = 1, 2, ..., m:
  w = Avⱼ
  for i = 1, ..., j:
    hᵢⱼ = vᵢᵀw
    w = w - hᵢⱼvᵢ
  hⱼ₊₁ⱼ = ||w||
  vⱼ₊₁ = w / hⱼ₊₁ⱼ
  
  Hⱼ = upper Hessenberg  (上 Hessenberg 矩阵)
  Solve: Hⱼy = θy
```

**优势**: 适用于非对称矩阵

## 5. 时间积分理论

### 5.1 Newmark 方法

**位移更新**:

```
uₙ₊₁ = uₙ + Δt vₙ + Δt²[(1/2 - β)aₙ + βaₙ₊₁]
```

**速度更新**:

```
vₙ₊₁ = vₙ + Δt[(1 - γ)aₙ + γaₙ₊₁]
```

**参数**:

- `β = 1/4`: 平均加速度（无条件稳定）
- `β = 1/6`: 线性加速度（条件稳定）
- `γ = 1/2`: 无阻尼

**有效刚度矩阵**:

```
K* = K + (1/(βΔt²))M + (γ/(βΔt))C
```

**有效载荷**:

```
F* = Fₙ₊₁ + M[(1/(βΔt²))uₙ + (1/(βΔt))vₙ + ((1-2β)/(2β))aₙ] +
     C[(γ/(βΔt))uₙ + ((γ/β)-1)vₙ + ((γ/(2β))-1)Δtaₙ]
```

### 5.2 α-OSP 方法（Generalized-α）

**参数**:

- `αₘ`: 质量参数
- `αᵢ`: 惯性参数
- `β`: Newmark 参数
- `γ`: Newmark 参数

**更新公式**:

```
uₙ₊₁ = uₙ + Δt vₙ + Δt²[(1/2 - β)aₙ + βaₙ₊₁]
vₙ₊₁ = vₙ + Δt[(1 - γ)aₙ + γaₙ₊₁]
aₙ₊₁₋αₘ = (1 - αₘ)aₙ₊₁ + αₘaₙ
uₙ₊₁₋αᵢ = (1 - αᵢ)uₙ₊₁ + αᵢuₙ
```

**优势**: 数值耗散可控，高频模态抑制

## 6. 稀疏矩阵理论

### 6.1 CSR 格式（Compressed Sparse Row）

**存储结构**:

- `val`: 非零元素值数组
- `col`: 列索引数组
- `row_ptr`: 行指针数组（`row_ptr[i]` 指向第 `i` 行的起始位置）

**示例**:

```
A = [
  1  0  2  0
  0  3  0  4
  5  0  6  7
  0  8  0  9
]

val = [1, 2, 3, 4, 5, 6, 7, 8, 9]
col = [0, 2, 1, 3, 0, 2, 3, 1, 3]
row_ptr = [0, 2, 4, 7, 9]
```

**优势**: 行访问高效，矩阵-向量乘法快

### 6.2 COO 格式（Coordinate）

**存储结构**:

- `val`: 非零元素值数组
- `row`: 行索引数组
- `col`: 列索引数组

**优势**: 构建简单，适合增量组装

**劣势**: 访问效率低（需排序）

### 6.3 稀疏矩阵-向量乘法（SpMV）

**CSR SpMV**:

```fortran
do i = 1, n_rows
  y(i) = 0.0
  do j = row_ptr(i), row_ptr(i+1) - 1
    y(i) = y(i) + val(j) * x(col(j))
  end do
end do
```

**优化**: 向量化、并行化、GPU 加速

## 7. 数值积分理论

### 7.1 高斯积分

**一维高斯积分**:

```
∫₋₁¹ f(ξ) dξ ≈ Σᵢ wᵢ f(ξᵢ)
```

**n点精度**: `2n-1` 次多项式精确

**常用积分点**:


| n   | ξᵢ                     | wᵢ       |
| --- | ---------------------- | -------- |
| 1   | 0                      | 2        |
| 2   | ±1/√3 ≈ ±0.577350      | 1        |
| 3   | 0, ±√(3/5) ≈ ±0.774597 | 8/9, 5/9 |


### 7.2 多维高斯积分

**矩形域**:

```
∫₋₁¹ ∫₋₁¹ f(ξ, η) dξ dη ≈ Σᵢ Σⱼ wᵢ wⱼ f(ξᵢ, ηⱼ)
```

**六面体域**:

```
∫₋₁¹ ∫₋₁¹ ∫₋₁¹ f(ξ, η, ζ) dξ dη dζ ≈ Σᵢ Σⱼ Σₖ wᵢ wⱼ wₖ f(ξᵢ, ηⱼ, ζₖ)
```
