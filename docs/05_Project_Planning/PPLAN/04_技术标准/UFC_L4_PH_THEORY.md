## 1. L4_PH 层概述

### 1.1 层级定位

**L4_PH = 物理计算层（Physics Computation Layer）**

**核心职责**:

- 从 L3_MD 读取模型描述数据（只读）
- 执行 Step 级物理计算
- 计算单元刚度矩阵、质量矩阵、载荷向量
- 评估材料本构关系（应力-应变）
- 处理接触和约束

**数据特征**:

- **Write-Once-Per-Step**（温数据）
- Step 边界创建/派生，Step 退出时可安全丢弃
- 不跨 Step 持久化

**依赖约束**:

- ✅ 允许 USE: L3_MD, L2_NM, L1_IF
- ❌ 禁止 USE: L5_RT, L6_AP

### 1.2 四条铁律


| 铁律              | 内容                                       |
| --------------- | ---------------------------------------- |
| **单向依赖**        | L4 → L3/L2/L1，禁止任何 L4 → L5/L6 反向调用       |
| **Step 级生命周期**  | L4 数据在 Step 边界创建/派生，Step 退出时可安全清理        |
| **只读 L3**       | L4 从 L3 容器只读获取模型描述，禁止修改 L3_MD 任何 Desc 字段 |
| **Ctx 不跨 Step** | PH_*_Ctx 类型生命周期严格限定在当前 Step 内            |

## 2. 有限元理论基础

### 2.1 弱形式与变分原理

**弱形式**（Weak Form）:

对于平衡方程:

```
∇·σ + b = 0  (在域 Ω 内)
```

弱形式为:

```
∫_Ω (∇·σ + b)·δu dΩ = 0
```

应用散度定理，得到:

```
∫_Ω σ:δε dΩ = ∫_Ω b·δu dΩ + ∫_∂Ω t·δu dS
```

其中:

- `σ`: 应力张量
- `ε`: 应变张量
- `b`: 体积力
- `t`: 表面力
- `δu`: 虚位移
- `δε`: 虚应变

### 2.2 离散化

**位移场离散化**:

```
u(x) = Σᵢ Nᵢ(x) uᵢ
```

其中:

- `Nᵢ(x)`: 形函数（Shape Function）
- `uᵢ`: 节点位移

**应变-位移关系**:

```
ε = B u
```

其中 `B` 是应变-位移矩阵（B矩阵）。

### 2.3 单元刚度矩阵

**单元刚度矩阵**:

```
Kᵉ = ∫_Ωᵉ Bᵀ D B dΩ
```

其中:

- `D`: 本构矩阵（应力-应变关系）
- `Ωᵉ`: 单元域

**数值积分**:

```
Kᵉ ≈ Σᵢ wᵢ Bᵀ(ξᵢ) D(ξᵢ) B(ξᵢ) |J(ξᵢ)|
```

其中:

- `wᵢ`: 积分权重
- `ξᵢ`: 积分点坐标（自然坐标）
- `J`: 雅可比矩阵

## 3. 单元理论

### 3.1 形函数（Shape Functions）

#### 一维线性单元（2节点）

**形函数**:

```
N₁(ξ) = (1 - ξ) / 2
N₂(ξ) = (1 + ξ) / 2
```

**自然坐标**: `ξ ∈ [-1, 1]`

#### 二维双线性单元（4节点四边形）

**形函数**:

```
N₁(ξ, η) = (1 - ξ)(1 - η) / 4
N₂(ξ, η) = (1 + ξ)(1 - η) / 4
N₃(ξ, η) = (1 + ξ)(1 + η) / 4
N₄(ξ, η) = (1 - ξ)(1 + η) / 4
```

**自然坐标**: `ξ, η ∈ [-1, 1]`

#### 三维三线性单元（8节点六面体，C3D8）

**形函数**:

```
Nᵢ(ξ, η, ζ) = (1 + ξᵢξ)(1 + ηᵢη)(1 + ζᵢζ) / 8
```

其中 `(ξᵢ, ηᵢ, ζᵢ)` 是节点 `i` 的自然坐标（±1）。

### 3.2 B矩阵（应变-位移矩阵）

**B矩阵定义**:

```
B = [B₁, B₂, ..., Bₙ]
```

对于三维问题，每个 `Bᵢ` 为:

```
Bᵢ = [
  ∂Nᵢ/∂x    0        0
  0        ∂Nᵢ/∂y    0
  0        0        ∂Nᵢ/∂z
  ∂Nᵢ/∂y  ∂Nᵢ/∂x   0
  0        ∂Nᵢ/∂z  ∂Nᵢ/∂y
  ∂Nᵢ/∂z  0        ∂Nᵢ/∂x
]
```

**坐标变换**:

```
[∂Nᵢ/∂x]   [∂Nᵢ/∂ξ]
[∂Nᵢ/∂y] = J⁻¹ [∂Nᵢ/∂η]
[∂Nᵢ/∂z]   [∂Nᵢ/∂ζ]
```

其中 `J` 是雅可比矩阵。

### 3.3 雅可比矩阵（Jacobian Matrix）

**雅可比矩阵**:

```
J = [
  ∂x/∂ξ  ∂y/∂ξ  ∂z/∂ξ
  ∂x/∂η  ∂y/∂η  ∂z/∂η
  ∂x/∂ζ  ∂y/∂ζ  ∂z/∂ζ
]
```

**坐标映射**:

```
x(ξ, η, ζ) = Σᵢ Nᵢ(ξ, η, ζ) xᵢ
y(ξ, η, ζ) = Σᵢ Nᵢ(ξ, η, ζ) yᵢ
z(ξ, η, ζ) = Σᵢ Nᵢ(ξ, η, ζ) zᵢ
```

**雅可比行列式**:

```
|J| = det(J)
```

**体积元**:

```
dΩ = |J| dξ dη dζ
```

### 3.4 单元质量矩阵

**一致质量矩阵**:

```
Mᵉ = ∫_Ωᵉ ρ Nᵀ N dΩ
```

**集中质量矩阵**（对角）:

```
Mᵢᵢ = ∫_Ωᵉ ρ Nᵢ dΩ
Mᵢⱼ = 0  (i ≠ j)
```

## 4. 材料本构理论

### 4.1 线弹性本构

**胡克定律**（Hooke's Law）:

```
σ = D ε
```

**各向同性弹性本构矩阵**:

```
D = E/(1+ν)(1-2ν) [
  1-ν  ν   ν   0    0    0
  ν   1-ν  ν   0    0    0
  ν   ν   1-ν  0    0    0
  0   0    0   (1-2ν)/2  0    0
  0   0    0   0    (1-2ν)/2  0
  0   0    0   0    0    (1-2ν)/2
]
```

其中:

- `E`: 杨氏模量（Young's Modulus）
- `ν`: 泊松比（Poisson's Ratio）

**体积模量**:

```
K = E / (3(1 - 2ν))
```

**剪切模量**:

```
G = E / (2(1 + ν))
```

### 4.2 弹塑性本构（J2 塑性）

**屈服准则**（Von Mises）:

```
f(σ, σᵧ) = √(3J₂) - σᵧ = 0
```

其中:

- `J₂ = (1/2) s:s`（第二应力不变量）
- `s = σ - (1/3)tr(σ)I`（偏应力）
- `σᵧ`: 屈服应力

**流动法则**（关联流动）:

```
dεᵖ = dλ (∂f/∂σ) = dλ (3s / (2σₑ))
```

其中 `dλ` 是塑性乘子。

**一致性条件**:

```
df = (∂f/∂σ):dσ + (∂f/∂σᵧ)dσᵧ = 0
```

**本构矩阵**（弹塑性）:

```
Dᵉᵖ = Dᵉ - (Dᵉ(∂f/∂σ) ⊗ (∂f/∂σ)Dᵉ) / (H + (∂f/∂σ):Dᵉ(∂f/∂σ))
```

其中 `H` 是硬化模量。

### 4.3 超弹性本构（Neo-Hookean）

**应变能函数**:

```
W = (μ/2)(I₁ - 3) + (K/2)(J - 1)²
```

其中:

- `I₁ = tr(C)`（第一不变量）
- `C = FᵀF`（右 Cauchy-Green 张量）
- `F`: 变形梯度
- `J = det(F)`（体积比）
- `μ`: 剪切模量
- `K`: 体积模量

**第二 Piola-Kirchhoff 应力**:

```
S = 2(∂W/∂C) = μ I + K(J - 1)JC⁻¹
```

## 5. 接触算法理论

### 5.1 接触检测

**主-从接触对**:

- **主面**（Master Surface）: 较粗网格
- **从面**（Slave Surface）: 较细网格

**穿透检测**:

```
g = (xₛ - xₘ)·nₘ
```

其中:

- `xₛ`: 从节点坐标
- `xₘ`: 主面上最近点坐标
- `nₘ`: 主面法向量

**接触条件**:

- `g < 0`: 穿透（需要约束）
- `g ≥ 0`: 分离（无约束）

### 5.2 接触约束（拉格朗日乘数法）

**约束方程**:

```
g = 0  (接触时)
```

**虚功**:

```
δWᶜ = λ δg
```

其中 `λ` 是接触力（拉格朗日乘数）。

**接触刚度矩阵**（罚函数法）:

```
Kᶜ = εᶜ n ⊗ n
```

其中 `εᶜ` 是罚参数。

## 6. 约束算法理论

### 6.1 多点约束（MPC）

**约束方程**:

```
Σᵢ cᵢ uᵢ = d
```

其中:

- `cᵢ`: 约束系数
- `uᵢ`: 节点位移
- `d`: 约束值

**拉格朗日乘数法**:

```
Kᵢⱼ = Kᵢⱼ + λᵢ cⱼ
Fᵢ = Fᵢ + λᵢ d
```

### 6.2 绑定约束（Tie Constraint）

**位移连续性**:

```
uₛ = uₘ  (在接触面上)
```

**约束力**:

```
Fᶜ = εᵗⁱᵉ (uₛ - uₘ)
```

其中 `εᵗⁱᵉ` 是绑定刚度（通常很大）。

## 7. 数值积分理论

### 7.1 高斯积分（Gauss Quadrature）

**一维高斯积分**:

```
∫₋₁¹ f(ξ) dξ ≈ Σᵢ wᵢ f(ξᵢ)
```

**n点高斯积分精度**: `2n-1` 次多项式精确

**常用积分点**:


| n   | ξᵢ         | wᵢ       |
| --- | ---------- | -------- |
| 1   | 0          | 2        |
| 2   | ±1/√3      | 1        |
| 3   | 0, ±√(3/5) | 8/9, 5/9 |


### 7.2 二维高斯积分

**矩形域**:

```
∫₋₁¹ ∫₋₁¹ f(ξ, η) dξ dη ≈ Σᵢ Σⱼ wᵢ wⱼ f(ξᵢ, ηⱼ)
```

**三角形域**（Dunavant 积分）:

```
∫_T f(ξ, η) dA ≈ Σᵢ wᵢ f(ξᵢ, ηᵢ)
```

### 7.3 三维高斯积分

**六面体域**:

```
∫₋₁¹ ∫₋₁¹ ∫₋₁¹ f(ξ, η, ζ) dξ dη dζ ≈ Σᵢ Σⱼ Σₖ wᵢ wⱼ wₖ f(ξᵢ, ηⱼ, ζₖ)
```

**常用积分方案**:

- **2×2×2**: 8点积分（完全积分，C3D8）
- **1×1×1**: 1点积分（减缩积分，C3D8R）
