## 3. 与总清单的关系

- **Txxxx** 编号与 [L3_L4_L5_pilot_f90任务清单.md](L3_L4_L5_pilot_f90任务清单.md) **全局一致**；波次子清单仅 **筛选行**，**不改编号**。
- 合并 PR 后，在 **总清单**或 **波次子清单** 中将对应 `- [ ]` 改为 `- [x]`（团队任选其一为主视图，避免双处漂移可只维护总清单中相关行）。
