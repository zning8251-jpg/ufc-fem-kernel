## 快速编号查询

### Solver维度 (1-based)

| 编号 | 名称 | 说明 |
|------|------|------|
| **1** | Standard | 隐式求解器，支持准静态/瞬态/线性，支持所有Physics |
| **2** | Explicit | 显式求解器，仅支持动力学，仅Physics=1 |
| **3** | Acoustic | 声学求解器，仅支持声学分析，仅Physics=4 |
| **4** | EM | 电磁求解器，仅支持电磁分析，仅Physics=5 |
| **5** | CFD | CFD求解器，支持流体分析，Physics=6,9,10 |

### Coupling维度 (1-based)

| 编号 | 名称 | 说明 | 迭代类型 |
|------|------|------|--------|
| **1** | OneShot | 单次求解，无耦合 | 无 |
| **2** | OneWay | 单向传递，单向耦合 | 单向 |
| **3** | Weak | 弱耦合，松耦合多场 | 松耦合 |
| **4** | Strong | 强耦合，紧耦合多场 | 紧耦合迭代 |

### Physics维度 (1-based) - 基本单场

| 编号 | 名称 | ABAQUS对标 | Solver |
|------|------|-----------|--------|
| **1** | Structure | 结构力学 | STD/EXP |
| **2** | Thermal | 热传导 | STD |
| **3** | Frequency | 频域分析 | STD |
| **4** | Acoustic | 声学 | ACO |
| **5** | EM | 电磁 | EM |
| **6** | Fluid | 流体(CFD) | CFD |

### Physics维度 (1-based) - 双场耦合

| 编号 | 名称 | 组成 | 推荐Coupling |
|------|------|------|------------|
| **7** | ThermalStruct | 热+结构 | Weak |
| **8** | ElectroStruct | 电+结构 | Strong |
| **9** | FluidStruct | 流体+结构(FSI) | Strong |
| **10** | FluidThermal | 流体+热 | Weak |

### Physics维度 (1-based) - 高阶与特殊

| 编号 | 名称 | 说明 |
|------|------|------|
| **11** | MultiField | 三场及以上耦合 |
| **12** | Special | 预留扩展 |

## 常用Group编码速查

### 结构分析 (Solver=1)

```
[1][1][1]  = Standard + OneShot + Structure      ⟵ PROC 1-10 (准静态)
[1][2][1]  = Standard + OneWay + Structure       (单向耦合)
[1][3][1]  = Standard + Weak + Structure         ⟵ PROC 32 (热结构,弱)
[1][4][1]  = Standard + Strong + Structure       (强耦合多场)
```

### 热分析 (Solver=1)

```
[1][1][2]  = Standard + OneShot + Thermal        (纯热分析)
[1][3][2]  = Standard + Weak + Thermal           (热耦合)
```

### 频域分析 (Solver=1)

```
[1][1][3]  = Standard + OneShot + Frequency      ⟵ PROC 20-22, 27 (模态)
```

### 显式动力学 (Solver=2)

```
[2][1][1]  = Explicit + OneShot + Structure      ⟵ PROC 11-19 (显式)
```

### 声学分析 (Solver=3)

```
[3][1][4]  = Acoustic + OneShot + Acoustic       ⟵ PROC 28 (稳态声学)
```

### 电磁分析 (Solver=4)

```
[4][1][5]  = EM + OneShot + EM                   ⟵ PROC 29 (稳态电磁)
```

### 耦合分析 (Solver=1)

```
[1][3][7]  = Standard + Weak + ThermalStruct     ⟵ PROC 32, 34 (热结构弱)
[1][4][8]  = Standard + Strong + ElectroStruct  ⟵ PROC 35 (电结构强)
[1][4][11] = Standard + Strong + MultiField     ⟵ PROC 33, 51 (多场强)
```

### CFD分析 (Solver=5)

```
[5][1][6]  = CFD + OneShot + Fluid               (纯流体)
[5][3][6]  = CFD + Weak + Fluid                  (流体+热,弱)
[5][3][10] = CFD + Weak + FluidThermal           (流体热耦合,弱)
[5][4][9]  = CFD + Strong + FluidStruct          (FSI,强,需STD)
[5][4][6]  = CFD + Strong + Fluid                (变工况流体)
```

## 编号转换规则

### 1-based (外部API) ↔ 0-based (内部实现)

```
internal = external - 1

示例:
[1][3][7] (1-based) → [0][2][6] (0-based)
[5][4][9] (1-based) → [4][3][8] (0-based)
```

### Group_ID编码公式

```
Group_ID_3D = Solver × 100 + Coupling × 10 + Physics
            = [1-5] × 100 + [1-4] × 10 + [1-12]

示例:
[1][3][7] → 100×1 + 10×3 + 7 = 137
[5][4][9] → 100×5 + 10×4 + 9 = 549
```

## 特殊场景

### 多求解器耦合 (需辅助求解器)

| 主Group | 辅助求解器 | 耦合算法 | 说明 |
|--------|----------|--------|------|
| [5][4][9] | Standard | Strong(FSI) | 流体-结构，需迭代耦合 |
| [5][3][10] | Standard/Thermal | Weak | 流体-热，需数据传递 |

### Geo处理 (通过Material族标记)

```
[1][1][1] + Material_Family=GEO
          ↓
岩土准静态分析，通过Material定义约束

[1][3][1] + Material_Family=GEO
          ↓
岩土瞬态分析，支持施工阶段
```

## 禁止组合速查

### 全局禁止

```
[*][2][2-12]        ❌ OneWay仅支持Structure
[1][1-4][4,5]       ❌ Standard不支持Acoustic/EM
[2][2-4][*]         ❌ Explicit仅支持OneShot
[3][2-4][*]         ❌ Acoustic仅支持OneShot
[4][2-4][*]         ❌ EM仅支持OneShot
[5][1-2][1-5,7,8]   ❌ CFD不支持Structure等单场
```

## 使用建议

1. **查询PROC映射**：用PROC-to-Group速查表直接找Group编码
2. **验证组合合法性**：查看合法组合矩阵，红色(❌)标记为禁止
3. **理解耦合策略**：结合Coupling维度含义理解分析方法
4. **跨求解器问题**：查看特殊场景部分了解辅助求解器需求
