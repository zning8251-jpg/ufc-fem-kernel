## 📋 核心规范总则

### 规范 1: 目录树路径映射

**原则**: 文件系统目录树 → 文档章节结构 → 一一对应

**格式**:
```
UFC/ufc_core/
├── L1_IF/                          → 文档 §1 (L1_IF 基础设施层)
│   ├── Base/                       → 文档 §1.1 (L1_IF/Base 域级)
│   │   ├── IF_Base_Utility.f90     → 文档 §1.1.1 (Base 域级内容详解)
│   │   ├── IF_Precision_Desc.f90   → 文档 §1.1.1.1 (Precision 子模块)
│   │   └── ...
│   ├── Error/                      → 文档 §1.2 (L1_IF/Error 域级)
│   └── ...
├── L2_NM/                          → 文档 §2 (L2_NM 数值算法层)
│   ├── Base/                       → 文档 §2.1 (L2_NM/Base 域级)
│   └── ...
├── L3_MD/                          → 文档 §3 (L3_MD 模型数据层)
└── ...
```

### 规范 2: 章节编号体系

**多层编号规则**:
```
§1                 — 顶层 (L1_IF 层级)
§1.1               — 域级 (L1_IF/Base)
§1.1.1             — 子域或模块组 (Base 下的功能模块)
§1.1.1.1           — 单个模块 (IF_Precision_Desc.f90)
§1.1.1.1.1         — 模块内的子功能 (TYPE 定义/SUBROUTINE 等)
```

**例子**:
```
§1.2.3             — L1_IF/Error 域级中的第 3 个子项
§3.2.4.2           — L3_MD/Material 域级中的第 4 个子域的第 2 个模块
```

### 规范 3: 命名约定

**文件命名** = **章节号** + **功能描述**:
```
✓ L1_IF_基础设施层_完整域级拆解_v1.0.md      → 对应 §1
✓ L3_MD_模型数据层_完整域级拆解_v1.0.md      → 对应 §3
✓ L5_RT_运行时协调层_完整域级拆解_v1.0.md    → 对应 §5
```

## 🎯 三、四型规范映射

### 3.1 四型 → 文档结构映射

每个模块在文档中的结构应遵循 **四型递进模式**:

```markdown
### §X.Y.Z.1 模块名 — Module Description

#### §X.Y.Z.1.1 Desc 类型 — 模型数据定义
TYPE :: MD_xxx_Desc
  ! 元数据、参数定义、Read-Once 配置
END TYPE
[详细字段说明表]

#### §X.Y.Z.1.2 State 类型 — 增量步状态
TYPE :: MD_xxx_State
  ! 状态变量、内变量、增量步读写
END TYPE
[详细字段说明表]

#### §X.Y.Z.1.3 Algo 类型 — 算法参数
TYPE :: MD_xxx_Algo
  ! 算法参数、数值策略、配置选项
END TYPE
[详细字段说明表]

#### §X.Y.Z.1.4 Ctx 类型 — 计算上下文
TYPE :: MD_xxx_Ctx
  ! 计算上下文、热路径、零 ALLOCATE 约束
END TYPE
[详细字段说明表]
```

### 3.2 模块级代码示例结构

```markdown
#### §X.Y.Z.1.5 关键接口函数

##### Desc 相关
- `SUBROUTINE init_xxx_desc(desc, params, status)` — 初始化
- `FUNCTION validate_xxx_desc(desc) RESULT(valid)` — 校验

##### State 相关
- `SUBROUTINE update_xxx_state(state, new_data, status)` — 状态更新
- `FUNCTION get_xxx_state_value(state, key) RESULT(value)` — 查询

##### Algo 相关
- `SUBROUTINE set_xxx_algorithm(algo, strategy, params)` — 算法设置

##### Ctx 相关
- `SUBROUTINE init_xxx_ctx(ctx, desc, state, status)` — 上下文初始化

#### §X.Y.Z.1.6 使用示例代码
[完整的 Fortran 代码示例，展示如何使用该模块]

#### §X.Y.Z.1.7 常见错误与规避
[表格形式的错误列表与解决方案]
```
