# L1_IF 基础设施层落地推演

> **文档位置**：`docs/05_Project_Planning/PPLAN/12_全栈域级落地矩阵/01_L1_IF_基础设施层落地推演.md`

## 总体定位
**L1_IF (Infrastructure)** 是一切业务的基础底座。在这个层级，没有任何物理概念（没有单元、没有节点、没有矩阵）。全域仅关注底层操作系统抽象、错误传播机制和内存管理。

---

### 域级编码： `IF_Memory` (内存与对象池域)

#### 1. 核心意图与完成体目标
- **设计意图**：统一接管 Fortran 内置的 `ALLOCATE`/`DEALLOCATE`，防止内存碎片与泄漏。
- **完成体目标**：所有上层核心数据（Desc/State）的开辟必须通过此域分配；大数组采用 Arena/Pool 机制进行快速复用。

#### 2. 合同卡设计 (Domain Contract)
- **输入依赖**：只读配置文件中关于最大可用内存的限额设定。
- **输出写回**：返回严格对齐的内存指针/句柄，并记录系统内存高水位线（High-water mark）。

#### 3. 十件套物理映射
| 模块类型 | 文件名推断 | 核心职责 |
| -------- | ---------- | -------- |
| _Def     | `IF_Memory_Def.f90` | 内存对齐常数（64-byte对齐等）、指针类型句柄定义 |
| _Ctx     | `IF_Memory_Ctx.f90` | 内存池的全局管理上下文（Arena 指针、使用量计数） |
| _Algo    | `IF_Memory.f90`（旧称 `IF_Memory_Algo.f90`）| 内存块分割、寻找空闲块、碎片整理算法 |
| _Brg     | `IF_Memory_Brg.f90` | 提供 `Memory_Alloc`、`Memory_Free` 宏门面 |
| _Dump    | `IF_Memory_Dump.f90`| 打印内存泄漏报告、未释放地址检查 |

#### 4. 打通思路与底层代码推断
所有L3/L4域的 `_Populate` 和运行时态创建必须调用 `IF_Memory_Brg`：
```fortran
! 底层代码推断：强类型句柄分配
SUBROUTINE IF_Memory_AllocReal2D(ptr, rows, cols, status)
    REAL(KIND=DP), POINTER :: ptr(:,:)
    INTEGER, INTENT(IN)    :: rows, cols
    TYPE(IF_Error_State), INTENT(OUT) :: status
    ! ... 调用底层 Algo 分配对齐内存
END SUBROUTINE
```

### 域级编码： `IF_Error` (错误传播域)

#### 1. 核心意图与完成体目标
- **设计意图**：废除传统的随意 `STOP` 中止或全局 `ierr` 变量。
- **完成体目标**：建立基于 `Result<T, E>` 理念的错误冒泡机制（Error Bubbling），确保如 L4 计算奇异时能优雅退回 L5 减小时间步。

#### 2. 合同卡设计 (Domain Contract)
- **输入依赖**：引发错误的源码行号、模块名、上下文特征码。
- **输出写回**：结构化的 `IF_Error_State` 错误栈对象。

#### 3. 十件套物理映射
| 模块类型 | 文件名推断 | 核心职责 |
| -------- | ---------- | -------- |
| _Def     | `IF_Error_Def.f90`  | 错误码枚举 (`ERR_SINGULAR`, `ERR_OOM`, `ERR_DIVERGE`) |
| _State   | `IF_Error_State.f90`| 单个错误链节点对象（包含错误码、信息字符串） |
| _Algo    | `IF_Error.f90`（旧称 `IF_Error_Algo.f90`） | 错误栈的回溯组装与堆栈展开解析逻辑 |
| _Brg     | `IF_Error_Brg.f90`  | 提供 `Error_Raise`, `Error_Check` 的上层宏接口 |

#### 4. 打通思路与底层代码推断
任何子程序的最后参数必须是 `err_status`：
```fortran
! 底层代码推断
SUBROUTINE RT_Solver_Run(..., err_stat)
    TYPE(IF_Error_State), INTENT(INOUT) :: err_stat
    
    ! 若下层单元计算抛错：
    CALL PH_Elem_Compute(..., err_stat)
    IF (IF_Error_IsBad(err_stat)) THEN
        CALL IF_Error_Wrap(err_stat, "RT_Solver_Run", "Element integration failed")
        RETURN
    END IF
END SUBROUTINE
```
