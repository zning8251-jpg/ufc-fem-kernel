## 0. 本 PR 的边界（避免「并行两整柱」失控）

本 PR 的**操作定义**不是「同时做完 P1 全族 + P2 全族」，而是锁定仓库里**已存在的一条生产热路径接缝**：

**L5 装配** `RT_Asm_GlobalStiffness` → **`PH_Element_Compute_Ke_Arg`** / `PH_Elem_Domain_Desc%Compute_Ke` → **材料属性注入**（`RT_Asm_Solv_KeArg_AttachMatProps` 等）→ **L3 网格/拓扑 SIO**（`MD_Mesh_GetElemConnect_*`）

即：**材料真源（Populate/槽位）如何进入单元 Ke 内核**，以及 **装配层如何编排该调用** — 首 PR 以 **对齐合同 + 形式（G1–G6）+ 接缝文档化** 为主；**不**以「新增全材料族 / 全单元族」为完成判据。

若需**全柱**收尾，在 **PR02+** 按同一检查表 **分别** 扩展 P1 或 P2 的未勾行。
