## 二、架构设计特点

### 2.1 严格三层嵌套

所有5个材料族都遵循相同的三层嵌套结构：

```fortran
TYPE :: MD_Mat_XXX_Desc
  INTEGER(i4) :: family_type      ! Level 1: 主族（11个）
  INTEGER(i4) :: sub_type         ! Level 2: 变体（47个已实现）
  INTEGER(i4) :: property_flags   ! Level 3: 属性标志
END TYPE
```

### 2.2 四类TYPE完整

所有5个材料族都实现了完整的四类TYPE系统：

| TYPE | 用途 | 层级 | 示例 |
|------|------|------|------|
| **Desc** | 静态描述符 | L3_MD | 材料参数、派生参数 |
| **State** | 运行时状态 | L4_PH/L5_RT | 应力/应变、内部变量 |
| **Algo** | 算法控制 | L4_PH | 积分方法、迭代参数 |
| **Ctx** | 迭代级工作区 | L4_PH | 临时变量、工作数组 |

### 2.3 L3/L4/L5三层贯通

所有5个材料族都实现了完整的三层贯通：

```
L3_MD (模型定义层)
    ↓ Populate (Brg)
L4_PH (物理计算层)
    ↓ Dispatch (RT)
L5_RT (运行时调度层)
```

### 2.4 命名规范统一

所有5个材料族都100%符合UFC_命名规范_v3.0：

- TYPE定义：`{层}_{域}_{功能}_{四型}`
- MODULE/文件：`{层}_{域}_{功能}[_{角色}].f90`
- 过程名：`{层}_{域}[_{功能}]_{Verb}[_{Object}]`

### 8.1 成功经验

1. **黄金模板威力巨大**：第一个材料族8小时，后续材料族30分钟
