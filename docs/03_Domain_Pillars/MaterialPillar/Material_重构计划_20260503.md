# UFC材料域deprecated文件重构计划

## 重构目标
将82个有用的deprecated文件中的代码合并到新的文件结构中，统一命名规范，然后删除旧文件。

## 命名规范
**层级_域缩写_材料族_模型名_角色后缀**

例如：
- `MD_Mat_Elas_Iso_Def.f90` - 各向同性弹性材料定义
- `MD_Mat_Plast_J2_Def.f90` - J2塑性材料定义
- `MD_Mat_Hyper_NeoHookean_Def.f90` - Neo-Hookean超弹性材料定义

## 重构步骤

### 第一步：检查重复文件
对于每个材料族，检查是否有重复的模型实现（如MD_Ela_Iso.f90和MD_Mat_Elas_Isotropic.f90）。如果有重复：
1. 比较两个文件的内容
2. 选择更完整的版本
3. 合并有用的代码

### 第二步：重构文件
对于每个deprecated文件：
1. 读取文件内容
2. 重构代码（统一命名规范）
3. 创建新文件（符合命名规范）
4. 验证新文件的正确性

### 第三步：删除旧文件
重构完毕后，删除deprecated目录中的旧文件。

### 第四步：验证完整性
验证所有材料模型都已经正确重构，没有遗漏。

## 下一步行动
1. 从最小的材料族开始（Elas - 9个文件）
2. 逐个检查重复文件
3. 重构文件
4. 验证完整性
5. 然后处理其他材料族
