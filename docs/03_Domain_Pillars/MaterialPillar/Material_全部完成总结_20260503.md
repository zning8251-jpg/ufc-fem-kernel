## 二、11个材料族详细介绍

### 2.1 弹性材料族（Elastic）

**变体（10种）**：
1. Isotropic（各向同性）
2. Orthotropic（正交各向异性）
3. Transversely Isotropic（横观各向同性）
4. Anisotropic（各向异性）
5. Porous（多孔）
6. Hypoelastic（亚弹性）
7. Shear（剪切）
8. Engineering Constants（工程常数）
9. Thermo-Elastic（热弹性）
10. Piezoelectric（压电）

**特点**：线性弹性，小变形，无内部变量

### 2.2 塑性材料族（Plastic）

**变体（12种）**：
1. J2 Isotropic（J2各向同性）
2. Kinematic Linear（线性随动硬化）
3. Kinematic Combined（混合硬化）
4. Hill（Hill各向异性）
5. Johnson-Cook（Johnson-Cook）
6. GTN（GTN多孔金属）
7. ORNL（ORNL本构）
8. Armstrong-Frederick（AF）
9. Chaboche（Chaboche多背应力）
10. Barlat（Barlat屈服准则）
11. Crystal Plasticity（晶体塑性）
12. J2 Tabular（J2表格硬化）

**特点**：弹塑性，返回映射算法，塑性内部变量

### 2.3 超弹性材料族（Hyperelastic）

**变体（11种）**：
1. Neo-Hookean
2. Mooney-Rivlin
3. Ogden
4. Yeoh
5. Arruda-Boyce
6. Van der Waals
7. Marlow
8. Polynomial
9. Reduced Polynomial
10. Gent
11. Blatz-Ko

**特点**：大变形，应变能函数，不变量

### 2.4 损伤材料族（Damage）

**变体（6种）**：
1. Ductile Damage（延性损伤）
2. Shear Damage（剪切损伤）
3. Brittle Damage（脆性损伤）
4. FLD（成形极限图）
5. CZM（内聚力模型）
6. Concrete Damage（混凝土损伤）

**特点**：损伤演化，失效判据，单元删除

### 2.5 蠕变材料族（Creep）

**变体（8种）**：
1. Power Law（幂律蠕变）
2. Time Hardening（时间硬化）
3. Strain Hardening（应变硬化）
4. Hyperbolic Sine（双曲正弦）
5. Norton-Bailey
6. Theta Projection
7. Anand（Anand粘塑性）
8. User-Defined（用户定义）

**特点**：时间相关，蠕变应变率，时间积分

### 2.6 粘弹性材料族（Viscoelastic）

**变体（4种）**：
1. Prony Series（Prony级数）
2. Maxwell（Maxwell模型）
3. Kelvin-Voigt（Kelvin-Voigt模型）
4. Standard Linear Solid（标准线性固体）

**特点**：时间相关，松弛模量，内部变量

### 2.7 岩土材料族（Geotechnical）

**变体（8种）**：
1. Drucker-Prager
2. Mohr-Coulomb
3. Cam-Clay
4. Modified Cam-Clay
5. Cap Model
6. Hoek-Brown
7. Jointed Rock
8. Crushable Foam

**特点**：压力相关，剪胀，硬化/软化

### 2.8 复合材料族（Composite）

**变体（5种）**：
1. Lamina（层合板）
2. Fabric（织物）
3. Hashin（Hashin失效准则）
4. Puck（Puck失效准则）
5. Progressive Damage（渐进损伤）

**特点**：各向异性，纤维方向，层间失效

### 2.9 热学材料族（Thermal）

**变体（3种）**：
1. Isotropic（各向同性）
2. Orthotropic（正交各向异性）
3. Temperature-Dependent（温度相关）

**特点**：热传导，热容，温度场

### 2.10 声学材料族（Acoustic）

**变体（2种）**：
1. Fluid（流体）
2. Solid（固体）

**特点**：声波传播，阻抗，吸声

### 2.11 用户定义材料族（User-Defined）

1. UMAT（用户材料子程序）
2. VUMAT（显式用户材料子程序）

**特点**：用户自定义，灵活扩展

## 三、架构设计特点

### 3.1 严格三层嵌套

所有11个材料族都遵循相同的三层嵌套结构：

```fortran
TYPE :: MD_Mat_XXX_Desc
  INTEGER(i4) :: family_type      ! Level 1: 主族（11个）
  INTEGER(i4) :: sub_type         ! Level 2: 变体（71个）
  INTEGER(i4) :: property_flags   ! Level 3: 属性标志（7个）
END TYPE
```

**三层嵌套枚举**：
- **Level 1**：11个主族（ELASTIC/PLASTIC/HYPERELASTIC/DAMAGE/CREEP/VISCOELASTIC/GEOTECHNICAL/COMPOSITE/THERMAL/ACOUSTIC/USER）
- **Level 2**：71个辅TYPE（每个主族2-12个变体）
- **Level 3**：7个属性标志（TEMP_DEP/FIELD_DEP/RATE_DEP/ANISO/COUPLED/NONLOCAL/USER）

### 3.2 四类TYPE完整

所有11个材料族都实现了完整的四类TYPE系统：

| TYPE | 用途 | 层级 | 示例 |
|------|------|------|------|
| **Desc** | 静态描述符 | L3_MD | 材料参数、派生参数 |
| **State** | 运行时状态 | L4_PH/L5_RT | 应力/应变、内部变量 |
| **Algo** | 算法控制 | L4_PH | 积分方法、迭代参数 |
| **Ctx** | 迭代级工作区 | L4_PH | 临时变量、工作数组 |

### 3.3 L3/L4/L5三层贯通

所有11个材料族都实现了完整的三层贯通：

```
L3_MD (模型定义层)
    ├─ 解析ABAQUS关键字
    ├─ 创建Desc
    ├─ 验证参数
    └─ 注册材料
    ↓ Populate (Brg)
L4_PH (物理计算层)
    ├─ 构建刚度矩阵
    ├─ 计算应力
    ├─ 计算切线
    └─ 更新状态
    ↓ Dispatch (RT)
L5_RT (运行时调度层)
    ├─ 路由调度
    ├─ State管理
    └─ Commit/Rollback
```

### 3.4 命名规范统一

所有11个材料族都100%符合UFC_命名规范_v3.0：

| 场景 | 公式 | 示例 |
|------|------|------|
| TYPE定义 | `{层}_{域}_{功能}_{四型}` | `MD_Mat_Elas_Desc` |
| MODULE/文件 | `{层}_{域}_{功能}[_{角色}].f90` | `MD_Mat_Elas_Core.f90` |
| 过程名 | `{层}_{域}[_{功能}]_{Verb}[_{Object}]` | `MD_Mat_Elas_Create_Isotropic` |

**角色后缀**：
- `_Def` - TYPE定义
- `_Core` - 核心实现
- `_Brg` - 桥接
- `_Eval` - 求值入口
- `_Compat` - 兼容层
