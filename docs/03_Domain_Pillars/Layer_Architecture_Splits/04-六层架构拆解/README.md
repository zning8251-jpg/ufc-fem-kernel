## 六层架构概述

### 依赖方向

L6 → L5 → L4 → L3 → L2 → L1（严格单向）

### 各层职责

- **L1_IF**：基础设施层，数值精度、内存管理、错误处理
- **L2_NM**：数值层，线性/非线性求解、特征值、GPU 内核
- **L3_MD**：模型层，几何、材料、截面、载荷、Step 定义
- **L4_PH**：物理层，本构关系、单元公式、接触、时间积分
- **L5_RT**：运行时层，Step 循环、求解器调度、装配、输出
- **L6_AP**：应用层，命令解析、前后处理接口、用户交互
