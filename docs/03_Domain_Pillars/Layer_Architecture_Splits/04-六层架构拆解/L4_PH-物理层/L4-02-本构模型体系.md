## 概述

L4_PH层的本构模型体系负责实现材料本构关系的物理计算，将L3_MD层定义的材料参数转换为应力-应变关系。这是有限元分析的核心物理计算，确保所有材料行为都有正确的本构实现。

**核心功能**：

- 本构评估统一接口
- 100+本构模型注册表
- UMAT/VUMAT桥接
- 一致切线刚度计算
- 状态变量管理

## 本构模型分类

### 弹性模型

**线性弹性**：

- 各向同性弹性（Isotropic Elasticity）
- 正交各向异性弹性（Orthotropic Elasticity）
- 横观各向同性弹性（Transversely Isotropic Elasticity）
- 完全各向异性弹性（Fully Anisotropic Elasticity）

**超弹性**：

- Neo-Hookean模型
- Mooney-Rivlin模型
- Ogden模型
- Yeoh模型
- Arruda-Boyce模型
- Van der Waals模型

### 塑性模型

**金属塑性**：

- Von Mises塑性
- Hill塑性
- Johnson-Cook模型
- Chaboche模型

**非金属塑性**：

- Drucker-Prager模型
- Mohr-Coulomb模型
- Cam-Clay模型
- Gurson模型
- CDP（混凝土损伤塑性）模型

### 损伤模型

- 延性损伤（Ductile Damage）
- 脆性损伤（Brittle Damage）
- 渐进损伤（Progressive Damage）
- 疲劳损伤（Fatigue Damage）

### 蠕变模型

- Norton蠕变
- Garofalo蠕变
- 时间硬化模型
- 应变硬化模型

### 粘弹性模型

- Prony级数模型
- Maxwell模型
- Kelvin模型

### 复合材料模型

- 层合板模型
- 纤维增强模型

## 本构评估统一接口

### 核心类型定义

```fortran
! L4_PH/Material/PH_MatConstitutive_Core.f90
MODULE PH_MatConstitutive_Core
    USE IF_Precision
    USE MD_Material_Def
    IMPLICIT NONE
  
    ! 材料点状态
    TYPE, PUBLIC :: MatPointState
        REAL(wp), ALLOCATABLE :: statev(:)        ! 状态变量
        REAL(wp), ALLOCATABLE :: statev_old(:)    ! 上一增量步状态变量
        REAL(wp) :: temperature = 0.0_wp          ! 温度
        REAL(wp) :: dt = 0.0_wp                   ! 时间增量
        REAL(wp) :: time = 0.0_wp                 ! 当前时间
    END TYPE MatPointState
  
    ! 材料点应力应变
    TYPE, PUBLIC :: MatPointStressStrain
        REAL(wp) :: strain(6) = 0.0_wp           ! Voigt应变
        REAL(wp) :: stress(6) = 0.0_wp            ! Voigt应力
        REAL(wp) :: tangent(6, 6) = 0.0_wp        ! 切线刚度矩阵
    END TYPE MatPointStressStrain
END MODULE PH_MatConstitutive_Core
```

```fortran
    TYPE(MatPointStressStrain), INTENT(IN) :: strain_stress
    REAL(wp), INTENT(OUT) :: tangent(6, 6)
```

## 本构模型注册表

### 注册表结构

```fortran
! L4_PH/Material/PH_MatConstitutive_Registry.f90
MODULE PH_MatConstitutive_Registry
    USE IF_Precision
    USE MD_Material_Def
    IMPLICIT NONE
  
    ! 模型注册表条目
    TYPE, PUBLIC :: ModelRegistryEntry
        INTEGER(i4) :: mat_id = 0                    ! 材料模型ID
        CHARACTER(LEN=64) :: mat_name                 ! 材料模型名称
        INTEGER(i4) :: num_props_required = 0         ! 所需参数数量
        INTEGER(i4) :: num_statev_required = 0        ! 所需状态变量数量
        LOGICAL :: is_implemented = .FALSE.           ! 是否已实现
        CHARACTER(LEN=256) :: description             ! 模型描述
    END TYPE ModelRegistryEntry
  
    ! 全局注册表（支持300个材料模型ID）
    TYPE(ModelRegistryEntry), SAVE :: g_model_registry(300)
  
    ! 注册所有模型
    SUBROUTINE PH_RegisterAllModels()
        ! Phase 1: 注册20个关键模型
        ! Phase 2: 扩展到100+模型
    END SUBROUTINE
  
    ! 查询模型信息
    FUNCTION PH_QueryModelInfo(mat_id) RESULT(info)
        INTEGER(i4), INTENT(IN) :: mat_id
        TYPE(ModelRegistryEntry) :: info
    END FUNCTION
  
    ! 调度应力更新
    SUBROUTINE PH_DispatchConstitutiveUpdate(mat_prop, mat_state, strain_stress, status)
        TYPE(MaterialPropertyDef), INTENT(IN) :: mat_prop
        TYPE(MatPointState), INTENT(INOUT) :: mat_state
        TYPE(MatPointStressStrain), INTENT(INOUT) :: strain_stress
        INTEGER(i4), INTENT(OUT) :: status
      
        SELECT CASE(mat_prop%mat_id)
        CASE(MAT_ID_ISOTROPIC_ELASTIC)
            CALL PH_UpdateStress_IsotropicElastic(mat_prop, mat_state, strain_stress, status)
        CASE(MAT_ID_VON_MISES)
            CALL PH_UpdateStress_VonMises(mat_prop, mat_state, strain_stress, status)
        CASE(MAT_ID_NEO_HOOKEAN)
            CALL PH_UpdateStress_NeoHookean(mat_prop, mat_state, strain_stress, status)
        ! ... 更多模型
        CASE DEFAULT
            status = STATUS_UNKNOWN_MATERIAL
        END SELECT
    END SUBROUTINE
END MODULE PH_MatConstitutive_Registry
```

### 材料模型ID分配

**Elastic (1-30)**：

- 1: 各向同性弹性
- 2-10: 正交各向异性、横观各向同性等
- 11-30: 其他弹性模型

**Plastic (31-100)**：

- 31: Von Mises
- 32-40: Hill、Drucker-Prager、Mohr-Coulomb等
- 41-60: Johnson-Cook、Chaboche等
- 61-80: Cam-Clay、Gurson、CDP等
- 81-100: 其他塑性模型

**Hyperelastic (101-130)**：

- 101: Neo-Hookean
- 102: Mooney-Rivlin
- 103: Ogden
- 104: Yeoh
- 105: Arruda-Boyce
- 106: Van der Waals
- 107-130: 其他超弹性模型

**Damage (131-160)**：

- 131-140: 延性损伤模型
- 141-150: 脆性损伤模型
- 151-160: 其他损伤模型

**Creep (161-180)**：

- 161: Norton蠕变
- 162-180: 其他蠕变模型

**Viscoelastic (181-200)**：

- 181: Prony级数
- 182-200: 其他粘弹性模型

**Composite (201-220)**：

- 201: 层合板
- 202-220: 其他复合材料模型

**User-Defined (251-300)**：

- 251-300: UMAT/VUMAT用户定义模型

## 本构模型实现示例

### Von Mises塑性实现

```fortran
! L4_PH/Material/PH_Mat_Plastic_VonMises.f90
MODULE PH_Mat_Plastic_VonMises
    USE IF_Precision
    USE PH_MatConstitutive_Core
    USE MD_Material_Def
    IMPLICIT NONE
  
    SUBROUTINE PH_UpdateStress_VonMises(mat_prop, mat_state, strain_stress, status)
        TYPE(MaterialPropertyDef), INTENT(IN) :: mat_prop
        TYPE(MatPointState), INTENT(INOUT) :: mat_state
        TYPE(MatPointStressStrain), INTENT(INOUT) :: strain_stress
        INTEGER(i4), INTENT(OUT) :: status
      
        REAL(wp) :: E, nu, sigma_y0, H
        REAL(wp) :: stress_trial(6), dev_stress(6)
        REAL(wp) :: s_norm, f_trial, delta_lambda
        REAL(wp) :: D_elastic(6, 6), mu
        REAL(wp) :: ep_eq_old, ep_eq_new
      
        ! 材料参数
        E = mat_prop%props(1)
        nu = mat_prop%props(2)
        sigma_y0 = mat_prop%props(3)
        H = mat_prop%props(4)  ! 硬化模量
      
        ! 等效塑性应变
        ep_eq_old = mat_state%statev(1)
      
        ! 1. 弹性预测
        CALL Compute_Isotropic_Elastic_Matrix(E, nu, D_elastic)
        stress_trial = strain_stress%stress + MATMUL(D_elastic, strain_stress%strain)
      
        ! 2. 计算偏应力
        dev_stress = Compute_Deviatoric_Stress(stress_trial)
      
        ! 3. 计算等效应力
        s_norm = SQRT(1.5_wp * DOT_PRODUCT(dev_stress(1:3), dev_stress(1:3)) + &
                      3.0_wp * DOT_PRODUCT(dev_stress(4:6), dev_stress(4:6)))
      
        ! 4. 当前屈服应力
        sigma_y = sigma_y0 + H * ep_eq_old
      
        ! 5. 检查屈服
        f_trial = s_norm - sigma_y
      
        IF (f_trial <= 0.0_wp) THEN
            ! 弹性状态
            strain_stress%stress = stress_trial
            strain_stress%tangent = D_elastic
        ELSE
            ! 塑性状态 - 径向返回
            mu = E / (2.0_wp * (1.0_wp + nu))
            delta_lambda = f_trial / (3.0_wp * mu + H)
          
            ! 更新应力
            strain_stress%stress = stress_trial - &
                                 delta_lambda * 3.0_wp * mu * dev_stress / s_norm
          
            ! 更新等效塑性应变
            ep_eq_new = ep_eq_old + delta_lambda
            mat_state%statev(1) = ep_eq_new
          
            ! 计算一致切线模量
            CALL Compute_Consistent_Tangent_Modulus(strain_stress%stress, dev_stress, &
                                                   s_norm, delta_lambda, &
                                                   E, nu, H, strain_stress%tangent)
        END IF
      
        status = IF_STATUS_OK
    END SUBROUTINE
END MODULE PH_Mat_Plastic_VonMises
```

### Neo-Hookean超弹性实现

```fortran
! L4_PH/Material/PH_Mat_Hyper_NeoHookean.f90
MODULE PH_Mat_Hyper_NeoHookean
    USE IF_Precision
    USE PH_MatConstitutive_Core
    USE MD_Material_Def
    IMPLICIT NONE
  
    SUBROUTINE PH_UpdateStress_NeoHookean(mat_prop, mat_state, strain_stress, status)
        TYPE(MaterialPropertyDef), INTENT(IN) :: mat_prop
        TYPE(MatPointState), INTENT(IN) :: mat_state
        TYPE(MatPointStressStrain), INTENT(INOUT) :: strain_stress
        INTEGER(i4), INTENT(OUT) :: status
      
        REAL(wp) :: C10, D1
        REAL(wp) :: F(3, 3), C(3, 3), I1, J
        REAL(wp) :: S(3, 3)  ! 第二Piola-Kirchhoff应力
      
        ! 材料参数
        C10 = mat_prop%props(1)
        D1 = mat_prop%props(2)
      
        ! 从应变计算变形梯度F（简化，实际需要从位移梯度计算）
        F = Compute_Deformation_Gradient(strain_stress%strain)
      
        ! 计算右Cauchy-Green张量
        C = MATMUL(TRANSPOSE(F), F)
      
        ! 计算第一不变量
        I1 = C(1,1) + C(2,2) + C(3,3)
      
        ! 计算体积比
        J = SQRT(DET(C))
      
        ! 计算第二Piola-Kirchhoff应力
        ! S = 2·∂W/∂C = 2·C10·I + 2·(1/D1)·(J-1)·(J/2)·C^(-1)
        CALL Compute_NeoHookean_Stress(C, C10, D1, I1, J, S)
      
        ! 计算Cauchy应力
        ! σ = (1/J)·F·S·F^T
        CALL Compute_Cauchy_Stress(F, S, J, strain_stress%stress)
      
        ! 计算一致切线模量
        CALL Compute_Hyperelastic_Tangent(F, C, C10, D1, I1, J, strain_stress%tangent)
      
        status = IF_STATUS_OK
    END SUBROUTINE
END MODULE PH_Mat_Hyper_NeoHookean
```

## UMAT/VUMAT桥接

### UMAT桥接接口

```fortran
! L4_PH/Material/PH_MatConstitutive_UMAT_Bridge.f90
MODULE PH_MatConstitutive_UMAT_Bridge
    USE IF_Precision
    USE PH_MatConstitutive_Core
    USE MD_Material_Def
    IMPLICIT NONE
  
    ! 调用用户UMAT子程序
    SUBROUTINE PH_UMAT_Bridge_Call(mat_prop, mat_state, strain_stress, status)
        TYPE(MaterialPropertyDef), INTENT(IN) :: mat_prop
        TYPE(MatPointState), INTENT(INOUT) :: mat_state
        TYPE(MatPointStressStrain), INTENT(INOUT) :: strain_stress
        INTEGER(i4), INTENT(OUT) :: status
      
        ! 1. 参数映射 L3_MD → UMAT
        CALL PH_UMAT_MapParams_L3toUMAT(mat_prop, PROPS, NPROPS)
        CALL PH_UMAT_MapStateVars_L3toUMAT(mat_state, STATEV, NSTATV)
      
        ! 2. 应力应变映射
        STRESS = strain_stress%stress
        DSTRAN = strain_stress%strain
      
        ! 3. 调用用户UMAT
        CALL UMAT(STRESS, STATEV, DDSDDE, SSE, SPD, SCD, &
                  RPL, DDSDDT, DRPLDE, DRPLDT, &
                  STRAN, DSTRAN, TIME, DTIME, TEMP, DTEMP, &
                  PREDEF, DPRED, CMNAME, NDI, NSHR, NTENS, &
                  NSTATV, PROPS, NPROPS, COORDS, DROT, PNEWDT, &
                  CELENT, DFGRD0, DFGRD1, NOEL, NPT, LAYER, KSPT, &
                  KSTEP, KINC)
      
        ! 4. 结果映射 UMAT → L4_PH
        strain_stress%stress = STRESS
        strain_stress%tangent = DDSDDE
        CALL PH_UMAT_MapStateVars_UMATtoL3(STATEV, NSTATV, mat_state)
      
        status = IF_STATUS_OK
    END SUBROUTINE
END MODULE PH_MatConstitutive_UMAT_Bridge
```

## 一致切线刚度计算

### 切线刚度的重要性

一致切线刚度矩阵 **D_e = ∂σ/∂ε** 对于Newton-Raphson方法的二次收敛至关重要。

**算法切线模量**：

```
D_alg = ∂σ_{n+1}/∂ε_{n+1}
```

**对于J2塑性**：

```
D_alg = D_el - (6G²/(3G+H)) · n⊗n - 
        (2G·Δλ/σ_eq_trial) · (I_dev - n⊗n)

其中：
- D_el：弹性本构矩阵
- G：剪切模量
- H：硬化模量
- n：流动方向
- I_dev：偏应力单位张量
```

## 物理一致性验证

### 客观性验证

**客观性原理**：本构关系必须满足客观性，即：

```
σ(Q·F) = Q·σ(F)·Q^T

其中Q为任意旋转张量
```

### 热力学一致性验证

**Clausius-Duhem不等式**：

```
σ : D - ρ(ψ̇ + ηθ̇) ≥ 0

其中：
- ψ：Helmholtz自由能
- η：熵
- θ：温度
```

### 材料对称性验证

- **各向同性**：材料性质在所有方向上相同
- **正交各向异性**：有三个相互垂直的对称面
- **横观各向同性**：有一个旋转对称轴

## 使用示例

```fortran
USE PH_MatConstitutive_Core
USE MD_Material_Def

TYPE(MaterialPropertyDef) :: mat_prop
TYPE(MatPointState) :: mat_state
TYPE(MatPointStressStrain) :: strain_stress
INTEGER(i4) :: status

! 初始化材料参数
mat_prop%mat_id = MAT_ID_VON_MISES
mat_prop%props = [200000.0_wp, 0.3_wp, 300.0_wp, 2000.0_wp]  ! E, nu, sigma_y0, H

! 初始化状态变量
CALL PH_InitializeStateVars(mat_prop, mat_state, status)

! 设置应变
strain_stress%strain = [0.001_wp, 0.0_wp, 0.0_wp, 0.0_wp, 0.0_wp, 0.0_wp]

! 更新应力
CALL PH_UpdateStress(mat_prop, mat_state, strain_stress, status)

IF (status == IF_STATUS_OK) THEN
    WRITE(*,*) '应力：', strain_stress%stress
    WRITE(*,*) '切线刚度矩阵已计算'
END IF
```
