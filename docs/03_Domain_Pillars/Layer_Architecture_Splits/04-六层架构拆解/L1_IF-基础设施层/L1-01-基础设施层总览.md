## 概述

L1_IF（Infrastructure Layer）是 UFC 六层架构的基础支撑层，提供数值精度、内存管理、错误处理、文件IO等基础设施服务。

**核心职责**：

- 数值精度定义和常量
- 内存管理和内存池
- 错误处理和错误码体系
- 文件IO和持久化
- 计时器和性能监控
- 依赖注入和配置管理

## 核心模块

### 精度与常量

- **IF_Prec**：数值精度定义（`wp`, `i4`等）
- **IF_Const**：数学常数和物理常数
- **IF_MathConst**：数学常数（π, e等）

### 内存管理

- **IF_Mem_Mgr**：内存管理器
- **IF_Workspace_Solver**：求解器工作空间管理
- **UF_StructMemPool**：结构化内存池（桥接到ufc_infra）

### 错误处理

- **IF_Err_Type**：错误类型定义
- **IF_Err_API**：错误处理API

### 文件IO

- **IF_FileIO**：文件读写接口
- **UF_StructFileManager**：结构化文件管理（桥接到ufc_infra）

### 计时器

- **IF_Timer**：计时器接口

## 与ufc_infra的桥接

L1_IF层通过Bridge模式桥接到外部基础设施库`ufc_infra`：

- **内存管理**：桥接到`UF_StructMemPool`
- **变量注册**：桥接到`UF_SymbolTableManager`
- **文件持久化**：桥接到`UF_StructFileManager`
- **统一数据平台**：桥接到`UF_DataPlatform`
