## 概述

L2_NM层的线性求解器负责求解线性方程组 **K·u = f**，这是有限元分析的核心数值操作。UFC提供直接求解器和迭代求解器两大类，支持CSR稀疏矩阵格式，并具备自动选择机制。

**核心功能**：

- 直接求解器（LDLT、Cholesky、LU分解）
- 迭代求解器（CG、GMRES、BiCGStab）
- 预条件器（ILU、AMG、Jacobi）
- 自动求解器选择
- CSR稀疏矩阵支持
- GPU加速支持

## 求解器分类

### 直接求解器（Direct Solvers）

**适用场景**：

- 小到中等规模问题（n < 10,000）
- 需要精确解
- 矩阵条件数良好

**核心算法**：

1. **LDLT分解**（对称矩阵）：

   ```
   A = L·D·L^T

   其中：
   - L：下三角矩阵
   - D：对角矩阵
   ```
2. **Cholesky分解**（对称正定矩阵）：

   ```
   A = L·L^T

   其中：
   - L：下三角矩阵
   ```
3. **LU分解**（一般矩阵）：

   ```
   A = L·U

   其中：
   - L：下三角矩阵
   - U：上三角矩阵
   ```

**求解流程**：

1. **分解阶段**：A → L·D·L^T（或L·U）
2. **前向替换**：L·y = b → y
3. **后向替换**：L^T·x = y（或U·x = y）→ x

### 迭代求解器（Iterative Solvers）

**适用场景**：

- 大规模问题（n > 10,000）
- 矩阵稀疏
- 可以接受近似解

**核心算法**：

1. **共轭梯度法（CG）**（对称正定矩阵）：

   ```
   初始化：r₀ = b - A·x₀, p₀ = r₀

   迭代：
     αₖ = (rₖ^T·rₖ) / (pₖ^T·A·pₖ)
     xₖ₊₁ = xₖ + αₖ·pₖ
     rₖ₊₁ = rₖ - αₖ·A·pₖ
     βₖ = (rₖ₊₁^T·rₖ₊₁) / (rₖ^T·rₖ)
     pₖ₊₁ = rₖ₊₁ + βₖ·pₖ
   ```
2. **GMRES**（一般矩阵）：

   ```
   使用Krylov子空间：
     Kₘ = span{r₀, A·r₀, A²·r₀, ..., A^(m-1)·r₀}

   在Kₘ中寻找最小残差解
   ```
3. **BiCGStab**（一般矩阵）：

   ```
   双共轭梯度稳定化方法
   适用于非对称矩阵
   ```

## 预条件器（Preconditioners）

预条件器用于加速迭代求解器的收敛。

### ILU预条件器

**不完全LU分解**：

```
M ≈ L·U

其中L和U是A的不完全分解
```

**类型**：

- **ILU(0)**：零填充，保持原始稀疏结构
- **ILUT**：阈值填充，控制填充量

### AMG预条件器

**代数多重网格**：

- 适用于大规模问题
- 自动构建多级网格
- 收敛速度快

### Jacobi预条件器

**对角预条件**：

```
M = diag(A)
```

- 计算简单
- 内存占用小
- 适用于对角占优矩阵

## CSR格式支持

所有求解器都支持CSR（Compressed Sparse Row）稀疏矩阵格式。

**CSR格式优势**：

- 内存效率高（稀疏度<10%时节省95%+内存）
- 矩阵向量乘法（SpMV）高效
- GPU友好

**核心操作**：

- **SpMV**：y = A·x（稀疏矩阵-向量乘法）
- **SpMV^T**：y = A^T·x（转置矩阵-向量乘法）

## 自动求解器选择

UFC提供自动求解器选择机制，根据矩阵特性自动选择最优求解器。

**选择策略**：

```fortran
! 自动选择求解器
if (n < size_threshold) then
    ! 小规模：使用直接法
    solver = SOLVER_DIRECT
else if (is_symmetric .and. is_positive_definite) then
    ! 对称正定：使用PCG
    solver = SOLVER_PCG
else if (is_symmetric) then
    ! 对称：使用LDLT
    solver = SOLVER_DIRECT
else
    ! 一般矩阵：使用GMRES或BiCGStab
    solver = SOLVER_GMRES
end if
```

**选择参数**：

- **size_threshold**：直接法阈值（默认5000）
- **is_symmetric**：矩阵对称性标志
- **is_positive_definite**：矩阵正定性标志

### 统一求解接口

```fortran
SUBROUTINE NM_Solve_Linear(csr_matrix, b, x, config, result, status)
    TYPE(NM_CSR_Type), INTENT(IN) :: csr_matrix    ! CSR矩阵
    REAL(wp), INTENT(IN) :: b(:)                   ! 右端向量
    REAL(wp), INTENT(INOUT) :: x(:)                ! 解向量
    TYPE(NM_LinearSolverConfig), INTENT(IN) :: config
    TYPE(NM_LinearSolverResult), INTENT(OUT) :: result
    INTEGER(i4), INTENT(OUT) :: status
  
    ! 自动选择或使用指定求解器
    SELECT CASE(config%solver_type)
    CASE(SOLVER_AUTO)
        CALL AutoSelectSolver(csr_matrix, config, result)
    CASE(SOLVER_DIRECT)
        CALL NM_Solve_Direct(csr_matrix, b, x, config, result, status)
    CASE(SOLVER_CG)
        CALL NM_Solve_CG(csr_matrix, b, x, config, result, status)
    CASE(SOLVER_GMRES)
        CALL NM_Solve_GMRES(csr_matrix, b, x, config, result, status)
    CASE DEFAULT
        status = STATUS_ERROR
    END SELECT
END SUBROUTINE NM_Solve_Linear
```

## 直接求解器实现

### LDLT分解（CSR格式）

```fortran
! L2_NM/Direct/NM_LDLT_CSR.f90
MODULE NM_LDLT_CSR
    USE IF_Precision
    USE NM_Sparse_Matrix_CSR
    IMPLICIT NONE
  
    TYPE, PUBLIC :: NM_LDLT_Factor_CSR
        TYPE(NM_CSR_Type) :: L                    ! 下三角矩阵（CSR）
        REAL(wp), ALLOCATABLE :: D(:)              ! 对角矩阵
        INTEGER(i4), ALLOCATABLE :: perm(:)        ! 置换向量
    END TYPE NM_LDLT_Factor_CSR
  
    ! LDLT分解
    SUBROUTINE NM_LDLT_Factorize_CSR(csr_matrix, factor, status)
        TYPE(NM_CSR_Type), INTENT(IN) :: csr_matrix
        TYPE(NM_LDLT_Factor_CSR), INTENT(OUT) :: factor
        INTEGER(i4), INTENT(OUT), OPTIONAL :: status
      
        ! 1. 符号分解（确定L的稀疏结构）
        CALL Symbolic_Factorization(csr_matrix, factor%L)
      
        ! 2. 数值分解（计算L和D）
        CALL Numeric_Factorization(csr_matrix, factor%L, factor%D)
    END SUBROUTINE
  
    ! LDLT求解
    SUBROUTINE NM_LDLT_Solve_CSR(factor, b, x, status)
        TYPE(NM_LDLT_Factor_CSR), INTENT(IN) :: factor
        REAL(wp), INTENT(IN) :: b(:)
        REAL(wp), INTENT(OUT) :: x(:)
        INTEGER(i4), INTENT(OUT), OPTIONAL :: status
      
        REAL(wp), ALLOCATABLE :: y(:), z(:)
      
        ! 前向替换：L·y = b
        CALL Forward_Substitution(factor%L, b, y)
      
        ! 对角求解：D·z = y
        z = y / factor%D
      
        ! 后向替换：L^T·x = z
        CALL Backward_Substitution(factor%L, z, x)
    END SUBROUTINE
END MODULE NM_LDLT_CSR
```

## 迭代求解器实现

### CG求解器（CSR格式）

```fortran
! L2_NM/Iterative/NM_CG_CSR.f90
MODULE NM_CG_CSR
    USE IF_Precision
    USE NM_Sparse_Matrix_CSR
    IMPLICIT NONE
  
    ! 共轭梯度法（CG）求解Ax = b（CSR格式）
    SUBROUTINE NM_CG_Solve_CSR(csr_matrix, b, x, tol, max_iter, status)
        TYPE(NM_CSR_Type), INTENT(IN) :: csr_matrix
        REAL(wp), INTENT(IN) :: b(:)
        REAL(wp), INTENT(INOUT) :: x(:)
        REAL(wp), INTENT(IN) :: tol
        INTEGER(i4), INTENT(IN) :: max_iter
        INTEGER(i4), INTENT(OUT) :: status
      
        REAL(wp), ALLOCATABLE :: r(:), p(:), Ap(:)
        REAL(wp) :: rho, rho_old, alpha, beta, residual_norm
        INTEGER(i4) :: iter
      
        ALLOCATE(r(SIZE(b)), p(SIZE(b)), Ap(SIZE(b)))
      
        ! 初始化
        CALL csr_matrix%matvec(x, Ap)
        r = b - Ap
        p = r
        rho = DOT_PRODUCT(r, r)
      
        ! CG迭代
        DO iter = 1, max_iter
            ! Ap = A * p
            CALL csr_matrix%matvec(p, Ap)
          
            ! alpha = rho / (p^T * Ap)
            alpha = rho / DOT_PRODUCT(p, Ap)
          
            ! x = x + alpha * p
            x = x + alpha * p
          
            ! r = r - alpha * Ap
            r = r - alpha * Ap
          
            ! 收敛检查
            residual_norm = SQRT(DOT_PRODUCT(r, r))
            IF (residual_norm < tol) THEN
                status = STATUS_CONVERGED
                EXIT
            END IF
          
            ! beta = rho_new / rho_old
            rho_old = rho
            rho = DOT_PRODUCT(r, r)
            beta = rho / rho_old
          
            ! p = r + beta * p
            p = r + beta * p
        END DO
      
        DEALLOCATE(r, p, Ap)
    END SUBROUTINE NM_CG_Solve_CSR
END MODULE NM_CG_CSR
```

## GPU加速支持

### GPU SpMV

```cuda
// L2_NM/GPU/NM_SpMV_GPU.cu
__global__ void SpMV_CSR_GPU(
    int n_rows,
    float* values,
    int* col_indices,
    int* row_ptr,
    float* x,
    float* y)
{
    int row = blockIdx.x * blockDim.x + threadIdx.x;
  
    if (row < n_rows) {
        float sum = 0.0f;
        int start = row_ptr[row];
        int end = row_ptr[row + 1];
      
        for (int j = start; j < end; j++) {
            sum += values[j] * x[col_indices[j]];
        }
      
        y[row] = sum;
    }
}
```

## 性能优化

### 矩阵重排序

**RCM（Reverse Cuthill-McKee）重排序**：

- 减少矩阵带宽
- 提高缓存命中率
- 加速直接求解器

**AMD（Approximate Minimum Degree）重排序**：

- 减少填充量
- 适用于稀疏矩阵分解

### 并行化

**OpenMP并行**：

- 矩阵向量乘法并行化
- 迭代求解器并行化

**MPI并行**：

- 大规模问题分布式求解
- 域分解方法

## 使用示例

```fortran
USE NM_Sparse_Matrix_CSR
USE NM_Linear_Solver

TYPE(NM_CSR_Type) :: K_csr
TYPE(NM_LinearSolverConfig) :: config
TYPE(NM_LinearSolverResult) :: result
REAL(wp), ALLOCATABLE :: b(:), x(:)
INTEGER(i4) :: status

! 初始化CSR矩阵
CALL K_csr%init(n, n, nnz_estimate)

! 配置求解器（自动选择）
config%solver_type = SOLVER_AUTO
config%tol = 1.0E-8_wp
config%max_iter = 1000

! 求解
CALL NM_Solve_Linear(K_csr, b, x, config, result, status)

IF (status == STATUS_CONVERGED) THEN
    WRITE(*,*) '收敛，迭代次数：', result%iterations
    WRITE(*,*) '残差范数：', result%residual_norm
END IF
```
