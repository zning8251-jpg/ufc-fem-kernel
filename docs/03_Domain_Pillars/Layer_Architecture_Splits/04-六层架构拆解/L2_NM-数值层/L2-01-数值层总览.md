## 概述

L2_NM（Numerical Methods Layer）是 UFC 六层架构的数值方法层，提供线性/非线性求解、特征值求解、CSR稀疏矩阵等核心数值算法。

**核心职责**：

- 线性求解器（直接/迭代）
- 非线性求解器（Newton-Raphson、弧长法）
- 特征值求解（Lanczos、Arnoldi）
- CSR稀疏矩阵存储和运算
- GPU加速内核

## 核心模块

### 线性求解器

- **NM_Solve_Linear**：线性求解器接口
- **NM_Solve_Direct**：直接求解器（LDLT、Cholesky）
- **NM_Solve_Iterative**：迭代求解器（CG、GMRES、BiCGStab）

### 非线性求解器

- **NM_Solve_Newton**：Newton-Raphson方法
- **NM_Solve_Riks**：Riks弧长法
- **NM_LineSearch**：线搜索算法

### 特征值求解

- **NM_Eigen_Lanczos**：Lanczos方法
- **NM_Eigen_Arnoldi**：Arnoldi方法
- **NM_Eigen_Subspace**：子空间迭代法

### CSR稀疏矩阵

- **NM_Sparse_Matrix_CSR**：CSR矩阵核心类型
- **NM_SpMV**：稀疏矩阵-向量乘法
- **NM_CSR_Assembly**：CSR矩阵组装

### GPU加速

- **NM_GPU_SpMV**：GPU稀疏矩阵-向量乘法
- **NM_GPU_Solver**：GPU求解器内核
