## 概述

L2_NM层的非线性求解器负责求解非线性平衡方程 **R(u) = F_int(u) - F_ext = 0**，这是有限元非线性分析的核心。UFC提供多种非线性求解方法，包括Newton-Raphson方法、弧长法、线搜索等。

**核心功能**：

- Newton-Raphson方法（完整、修正、准Newton）
- Riks弧长法（追踪后屈曲路径）
- 线搜索（改善收敛性）
- 收敛判断（力收敛、位移收敛、能量收敛）
- 发散检测和增量切割

## 非线性平衡方程

**基本形式**：

```
R(u) = F_int(u) - F_ext = 0

其中：
- R(u)：残差向量
- F_int(u)：内力向量（依赖于位移u）
- F_ext：外载荷向量
```

**线性化**：

```
R(u + Δu) ≈ R(u) + K_T · Δu = 0

其中：
- K_T = ∂F_int/∂u：切线刚度矩阵
- Δu：位移增量
```

## Newton-Raphson方法

### 完整Newton-Raphson方法

**特点**：每次迭代都更新切线刚度矩阵

**算法流程**：

```
初始化：u₀ = u_n

for iteration = 1 to max_iterations:
    1. 计算残差：R = F_ext - F_int(u)
    2. 计算切线刚度：K_T = ∂F_int/∂u
    3. 求解线性系统：K_T · Δu = R
    4. 更新位移：u = u + Δu
    5. 收敛检查：if ||R|| < tol_force and ||Δu|| < tol_disp: break
```

**优点**：
- 二次收敛速度
- 收敛稳定

**缺点**：
- 需要每次迭代计算切线刚度矩阵
- 计算成本高

### 修正Newton-Raphson方法

**特点**：使用初始切线刚度矩阵，不更新

**算法流程**：

```
初始化：u₀ = u_n, K_T₀ = K_T(u_n)

for iteration = 1 to max_iterations:
    1. 计算残差：R = F_ext - F_int(u)
    2. 求解线性系统：K_T₀ · Δu = R  （使用初始刚度）
    3. 更新位移：u = u + Δu
    4. 收敛检查：if ||R|| < tol: break
```

**优点**：
- 不需要重新计算切线刚度矩阵
- 计算成本低

**缺点**：
- 收敛速度较慢
- 可能需要更多迭代次数

### 准Newton方法（BFGS）

**特点**：使用近似切线刚度矩阵，通过BFGS更新

**算法流程**：

```
初始化：u₀ = u_n, H₀ = K_T⁻¹(u_n)

for iteration = 1 to max_iterations:
    1. 计算残差：R = F_ext - F_int(u)
    2. 求解：Δu = H · R  （使用近似逆矩阵）
    3. 更新位移：u = u + Δu
    4. BFGS更新：H = BFGS_Update(H, Δu, ΔR)
    5. 收敛检查：if ||R|| < tol: break
```

**优点**：
- 不需要计算切线刚度矩阵
- 收敛速度介于完整和修正Newton-Raphson之间

**缺点**：
- 需要存储和更新近似矩阵
- 内存占用较大

## Riks弧长法

**目的**：追踪后屈曲路径，处理载荷-位移曲线中的极值点和分叉点

**增广系统**：

```
R(u, λ) = F_int(u) - λ·F_ref = 0

约束条件：||Δu||² + α²·(Δλ)² = Δs²

其中：
- λ：载荷因子
- F_ref：参考载荷向量
- Δs：弧长增量
- α：缩放因子
```

**增广线性系统**：

```
[K_T  -F_ref] [Δu  ]   [-R      ]
[2Δu  2α²Δλ] [Δλ  ] = [-约束残差]
```

**算法流程**：

```
初始化：u₀ = u_n, λ₀ = λ_n

for iteration = 1 to max_iterations:
    1. 计算切线刚度：K_T = ∂F_int/∂u
    2. 计算残差：R = λ·F_ref - F_int(u)
    3. 求解增广系统：得到Δu和Δλ
    4. 更新：u = u + Δu, λ = λ + Δλ
    5. 收敛检查：if ||R|| < tol: break
```

**应用场景**：
- 屈曲分析
- 后屈曲分析
- 载荷-位移曲线有极值点的问题

## 线搜索（Line Search）

**目的**：改善Newton-Raphson方法的收敛性，特别是在高度非线性问题中

**方法**：

在Newton-Raphson迭代中，不直接使用完整的位移增量，而是使用缩放后的增量：

```
u^(i+1) = u^(i) + s·Δu^(i)

其中：
- s：线搜索参数（0 < s ≤ 1）
- 通过最小化残差范数确定：minimize ||R(u^(i) + s·Δu^(i))||
```

**线搜索算法**：

```
1. 初始步长：s = 1.0
2. 检查残差：if ||R(u + s·Δu)|| < ||R(u)||: 接受该步长
3. 回溯：if 残差增大: s = β·s（通常β = 0.5）
4. 迭代：重复步骤2-3，直到满足收敛条件或达到最大迭代次数
```

**线搜索类型**：

- **Secant方法**：使用割线法寻找最优步长
- **Bisection方法**：二分法寻找最优步长
- **Armijo条件**：满足Armijo条件即可

## 收敛判断

### 力收敛（Force Convergence）

```
||R|| / ||F_ref|| < tol_force

其中：
- ||R||：残差向量范数
- ||F_ref||：参考载荷向量范数
- tol_force：力收敛容差（默认1.0E-6）
```

### 位移收敛（Displacement Convergence）

```
||Δu|| / ||u|| < tol_disp

其中：
- ||Δu||：位移增量范数
- ||u||：当前位移范数
- tol_disp：位移收敛容差（默认1.0E-6）
```

### 能量收敛（Energy Convergence）

```
Δu^T · R / (Δu₀^T · R₀) < tol_energy

其中：
- Δu^T · R：能量增量
- Δu₀^T · R₀：初始能量增量
- tol_energy：能量收敛容差（默认1.0E-6）
```

**收敛条件**：通常需要同时满足力收敛和位移收敛

## 发散检测和增量切割

### 发散检测

**发散判据**：

```
if ||R|| > diverge_ratio · ||R₀||:
    发散，需要切割增量
```

**最大残差限制**：

```
if ||R|| > max_residual:
    发散，停止计算
```

### 增量切割（Increment Cutting）

**目的**：当迭代发散时，减小增量步长

**算法**：

```
if 发散:
    1. 恢复上一增量步的状态
    2. 减小增量步长：Δs_new = β·Δs_old（通常β = 0.5）
    3. 重新开始增量步
    4. 如果切割次数 > max_cuts: 停止计算
```

## 统一接口设计

### 求解器配置

```fortran
TYPE, PUBLIC :: NM_NonlinearSolverConfig
    INTEGER(i4) :: method_type = METHOD_FULL_NR      ! 方法类型
    INTEGER(i4) :: max_iterations = 20               ! 最大迭代次数
    REAL(wp) :: tolerance_force = 1.0E-6_wp          ! 力收敛容差
    REAL(wp) :: tolerance_displacement = 1.0E-6_wp  ! 位移收敛容差
    REAL(wp) :: tolerance_energy = 1.0E-6_wp        ! 能量收敛容差
    LOGICAL :: use_line_search = .TRUE.              ! 使用线搜索
    REAL(wp) :: line_search_tolerance = 0.9_wp       ! 线搜索容差
    INTEGER(i4) :: line_search_max_iter = 10         ! 线搜索最大迭代
    REAL(wp) :: diverge_ratio = 1.0E10_wp            ! 发散判据
    INTEGER(i4) :: max_cuts = 5                      ! 最大切割次数
END TYPE NM_NonlinearSolverConfig
```

### 求解结果

```fortran
TYPE, PUBLIC :: NM_NonlinearSolverResult
    INTEGER(i4) :: iterations = 0                    ! 迭代次数
    LOGICAL :: converged = .FALSE.                   ! 是否收敛
    REAL(wp) :: residual_norm = 0.0_wp             ! 最终残差范数
    REAL(wp) :: displacement_norm = 0.0_wp         ! 位移增量范数
    REAL(wp) :: energy_increment = 0.0_wp           ! 能量增量
    INTEGER(i4) :: line_search_iterations = 0       ! 线搜索迭代次数
    INTEGER(i4) :: cuts = 0                          ! 增量切割次数
END TYPE NM_NonlinearSolverResult
```

### 统一求解接口

```fortran
SUBROUTINE NM_Solve_Nonlinear(residual_func, jacobian_func, u0, config, result, status)
    PROCEDURE(ResidualFunc) :: residual_func         ! 残差函数
    PROCEDURE(JacobianFunc) :: jacobian_func         ! 雅可比函数
    REAL(wp), INTENT(INOUT) :: u0(:)                 ! 初始位移
    TYPE(NM_NonlinearSolverConfig), INTENT(IN) :: config
    TYPE(NM_NonlinearSolverResult), INTENT(OUT) :: result
    INTEGER(i4), INTENT(OUT) :: status
    
    SELECT CASE(config%method_type)
    CASE(METHOD_FULL_NR)
        CALL NM_Solve_FullNewton(residual_func, jacobian_func, u0, config, result, status)
    CASE(METHOD_MODIFIED_NR)
        CALL NM_Solve_ModifiedNewton(residual_func, jacobian_func, u0, config, result, status)
    CASE(METHOD_RIKS)
        CALL NM_Solve_Riks(residual_func, jacobian_func, u0, config, result, status)
    CASE DEFAULT
        status = STATUS_ERROR
    END SELECT
END SUBROUTINE NM_Solve_Nonlinear
```

## 实现示例

### 完整Newton-Raphson实现

```fortran
! L2_NM/Nonlinear/NM_Solve_FullNewton.f90
MODULE NM_Solve_FullNewton
    USE IF_Precision
    USE NM_Linear_Solver
    IMPLICIT NONE
    
    SUBROUTINE NM_Solve_FullNewton(residual_func, jacobian_func, u0, config, result, status)
        PROCEDURE(ResidualFunc) :: residual_func
        PROCEDURE(JacobianFunc) :: jacobian_func
        REAL(wp), INTENT(INOUT) :: u0(:)
        TYPE(NM_NonlinearSolverConfig), INTENT(IN) :: config
        TYPE(NM_NonlinearSolverResult), INTENT(OUT) :: result
        INTEGER(i4), INTENT(OUT) :: status
        
        REAL(wp), ALLOCATABLE :: R(:), K_T(:,:), du(:)
        REAL(wp) :: residual_norm, disp_norm
        INTEGER(i4) :: iter
        
        ALLOCATE(R(SIZE(u0)), K_T(SIZE(u0), SIZE(u0)), du(SIZE(u0)))
        
        DO iter = 1, config%max_iterations
            ! 1. 计算残差
            CALL residual_func(u0, R)
            
            ! 2. 计算切线刚度矩阵
            CALL jacobian_func(u0, K_T)
            
            ! 3. 求解线性系统
            CALL NM_Solve_Linear(K_T, R, du, config%linear_config, status)
            
            ! 4. 线搜索（可选）
            IF (config%use_line_search) THEN
                CALL NM_LineSearch(u0, du, R, residual_func, config, u0, status)
            ELSE
                u0 = u0 + du
            END IF
            
            ! 5. 收敛检查
            residual_norm = NORM2(R)
            disp_norm = NORM2(du)
            
            IF (residual_norm < config%tolerance_force .AND. &
                disp_norm < config%tolerance_displacement) THEN
                result%converged = .TRUE.
                result%iterations = iter
                EXIT
            END IF
            
            ! 6. 发散检测
            IF (residual_norm > config%diverge_ratio * result%residual_norm) THEN
                status = STATUS_DIVERGED
                EXIT
            END IF
        END DO
        
        DEALLOCATE(R, K_T, du)
    END SUBROUTINE
END MODULE NM_Solve_FullNewton
```

### Riks弧长法实现

```fortran
! L2_NM/Nonlinear/NM_Solve_Riks.f90
MODULE NM_Solve_Riks
    USE IF_Precision
    USE NM_Linear_Solver
    IMPLICIT NONE
    
    SUBROUTINE NM_Solve_Riks(residual_func, jacobian_func, u0, lambda0, config, result, status)
        PROCEDURE(ResidualFunc) :: residual_func
        PROCEDURE(JacobianFunc) :: jacobian_func
        REAL(wp), INTENT(INOUT) :: u0(:)
        REAL(wp), INTENT(INOUT) :: lambda0
        TYPE(NM_NonlinearSolverConfig), INTENT(IN) :: config
        TYPE(NM_NonlinearSolverResult), INTENT(OUT) :: result
        INTEGER(i4), INTENT(OUT) :: status
        
        REAL(wp), ALLOCATABLE :: R(:), K_T(:,:), F_ref(:)
        REAL(wp), ALLOCATABLE :: aug_system(:,:), aug_rhs(:), aug_sol(:)
        REAL(wp) :: delta_s, alpha
        INTEGER(i4) :: n, iter
        
        n = SIZE(u0)
        ALLOCATE(R(n), K_T(n,n), F_ref(n))
        ALLOCATE(aug_system(n+1,n+1), aug_rhs(n+1), aug_sol(n+1))
        
        ! 初始化参考载荷
        F_ref = config%reference_load
        
        DO iter = 1, config%max_iterations
            ! 1. 计算切线刚度矩阵
            CALL jacobian_func(u0, K_T)
            
            ! 2. 计算残差
            CALL residual_func(u0, R)
            R = lambda0 * F_ref - R
            
            ! 3. 构建增广系统
            aug_system(1:n, 1:n) = K_T
            aug_system(1:n, n+1) = -F_ref
            aug_system(n+1, 1:n) = 2.0_wp * u0
            aug_system(n+1, n+1) = 2.0_wp * alpha**2 * lambda0
            
            aug_rhs(1:n) = -R
            aug_rhs(n+1) = -(delta_s**2 - DOT_PRODUCT(u0, u0) - alpha**2 * lambda0**2)
            
            ! 4. 求解增广系统
            CALL NM_Solve_Linear(aug_system, aug_rhs, aug_sol, config%linear_config, status)
            
            ! 5. 更新
            u0 = u0 + aug_sol(1:n)
            lambda0 = lambda0 + aug_sol(n+1)
            
            ! 6. 收敛检查
            IF (NORM2(R) < config%tolerance_force) THEN
                result%converged = .TRUE.
                result%iterations = iter
                EXIT
            END IF
        END DO
        
        DEALLOCATE(R, K_T, F_ref, aug_system, aug_rhs, aug_sol)
    END SUBROUTINE
END MODULE NM_Solve_Riks
```

## 使用示例

```fortran
USE NM_Nonlinear_Solver

TYPE(NM_NonlinearSolverConfig) :: config
TYPE(NM_NonlinearSolverResult) :: result
REAL(wp), ALLOCATABLE :: u(:)
INTEGER(i4) :: status

! 配置求解器
config%method_type = METHOD_FULL_NR
config%max_iterations = 20
config%tolerance_force = 1.0E-6_wp
config%tolerance_displacement = 1.0E-6_wp
config%use_line_search = .TRUE.

! 求解
CALL NM_Solve_Nonlinear(ComputeResidual, ComputeJacobian, u, config, result, status)

IF (result%converged) THEN
    WRITE(*,*) '收敛，迭代次数：', result%iterations
    WRITE(*,*) '残差范数：', result%residual_norm
ELSE
    WRITE(*,*) '未收敛，迭代次数：', result%iterations
END IF
```
