## L2_NM 层概述

### 核心功能

- 线性求解（直接/迭代/CG）
- 非线性（NR、弧长、线搜索）
- 特征值（Lanczos、Arnoldi、子空间迭代）
- GPU 内核（cuBLAS 等）

### 关键模块

- 线性求解器模块
- 非线性求解器模块
- 特征值求解器模块
- CSR 稀疏矩阵模块
