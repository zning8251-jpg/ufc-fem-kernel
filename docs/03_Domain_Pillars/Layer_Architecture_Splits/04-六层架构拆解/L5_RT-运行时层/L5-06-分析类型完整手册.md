## 二、静力学分析

### 2.1 非线性静力学分析（*STATIC）

#### 2.1.1 基本理论

**平衡方程**：

```
K(u) · u = F_ext

其中：
- K(u): 切线刚度矩阵（依赖于位移）
- u: 位移向量
- F_ext: 外载荷向量
```

**Newton-Raphson迭代**：

```
K_T^(i) · Δu^(i) = R^(i)
u^(i+1) = u^(i) + Δu^(i)

其中：
- K_T^(i): 第i次迭代的切线刚度矩阵
- R^(i) = F_ext - F_int^(i): 残差向量
- F_int^(i): 第i次迭代的内力向量
```

#### 2.1.2 收敛准则

**力收敛**：

```
|||R|| < ε_f · ||F_ext||

其中：
- ||R||: 残差向量的范数
- ||F_ext||: 外载荷向量的范数
- ε_f: 力收敛容差（默认：1.0e-3）
```

**位移收敛**：

```
|||Δu|| < ε_u · ||Δu_max||

其中：
- ||Δu||: 位移增量的范数
- ||Δu_max||: 最大位移增量
- ε_u: 位移收敛容差（默认：1.0e-3）
```

**能量收敛**：

```
||Δu · R| < ε_e · |Δu_0 · R_0|

其中：
- Δu · R: 能量增量
- Δu_0 · R_0: 初始能量增量
- ε_e: 能量收敛容差（默认：1.0e-6）
```

#### 2.1.3 UFC实现

```fortran
! L5_RT 非线性静力求解
module RT_Solver_Static
  public :: RT_SolveStatic
  
  subroutine RT_SolveStatic(model, step, results)
    type(MD_Model), intent(in) :: model
    type(StepDefinition), intent(in) :: step
    type(AnalysisResults), intent(out) :: results
  
    ! 1. 组装全局刚度矩阵 K
    ! 2. 组装载荷向量 f
    ! 3. 施加边界条件
    ! 4. Newton-Raphson迭代：
    !    for iteration = 1 to max_iterations:
    !       1. 计算残差：R = F_ext - F_int(u)
    !       2. 计算切线刚度：K_T = ∂F_int/∂u
    !       3. 求解：K_T · Δu = R
    !       4. 更新：u = u + Δu
    !       5. 检查收敛：||R|| < tolerance
    !       6. 如果收敛，退出；否则继续迭代
    ! 5. 后计算：σ = D·B·u, ε = B·u
    ! 6. 存储结果
  end subroutine
end module
```

### 2.2 Riks弧长法（*STATIC, RIKS）

#### 2.2.1 基本理论

**弧长法控制方程**：


\mathbf{R}(\mathbf{u}, \lambda) = \mathbf{F}*{int}(\mathbf{u}) - \lambda \cdot \mathbf{F}*{ref} = \mathbf{0}



g(\mathbf{u}, \lambda) = \Delta \mathbf{u}^2 + \alpha \cdot (\Delta \lambda)^2 - (\Delta l)^2 = 0


其中：

- $\lambda$: 载荷因子
- $\mathbf{F}_{ref}$: 参考载荷向量
- $\alpha$: 弧长参数
- $\Delta l$: 弧长增量

**Crisfield球形方法**（$\alpha = 0$）：


\Delta \mathbf{u}^2 = (\Delta l)^2


**Riks圆柱形方法**（$\alpha = 1$）：


\Delta \mathbf{u}^2 + (\Delta \lambda)^2 = (\Delta l)^2


#### 2.2.2 UFC实现

```fortran
! L5_RT Riks弧长法求解

  subroutine RT_SolveRiks(model, step, results)

    type(StepDefinition), intent(in) :: step
    type(AnalysisResults), intent(out) :: results
  
    ! Riks弧长法迭代：
    ! 1. 预测步：沿切线方向预测
    ! 2. 修正步：Newton-Raphson迭代修正
    ! 3. 弧长控制：自适应调整弧长增量

  end subroutine
end module
```

## 三、动力学分析

### 3.1 隐式动力分析（*DYNAMIC）

#### 3.1.1 基本理论

**运动方程**：

```
M · a + C · v + K · u = F(t)
```

其中：

- M: 质量矩阵
- C: 阻尼矩阵
- K: 刚度矩阵
- u, v, a: 位移、速度、加速度向量
- F(t): 时变载荷向量

**Newmark-β方法**：

```
u_{n+1} = u_n + Δt · v_n + (Δt)²/2 · [(1-2β)a_n + 2βa_{n+1}]
v_{n+1} = v_n + Δt · [(1-γ)a_n + γa_{n+1}]
```

其中：

- β = 1/4: 平均加速度法（无条件稳定）
- γ = 1/2: 线性加速度法

**HHT-α方法**（数值阻尼）：

```
M · a_{n+1} + (1+α)·C·v_{n+1} - α·C·v_n + (1+α)·K·u_{n+1} - α·K·u_n = F_{n+1+α}
```

其中：

- α ∈ [-1/3, 0]: 数值阻尼参数
- α = 0: 退化为Newmark方法

#### 3.1.2 UFC实现

```fortran
! L5_RT 隐式动力分析求解
module RT_Solver_Dynamic
  public :: RT_SolveDynamic
  
  subroutine RT_SolveDynamic(model, step, results)
    type(MD_Model), intent(in) :: model
    type(StepDefinition), intent(in) :: step
    type(AnalysisResults), intent(out) :: results
  
    ! 时间积分循环：
    ! for t = t_start to t_end:
    !   1. 计算有效刚度：K_eff = K + a₀·M + a₁·C
    !   2. 计算有效载荷：f_eff = f + M·(a₀·u_n + a₂·v_n + a₃·a_n) + ...
    !   3. 求解：K_eff · u_{n+1} = f_eff
    !   4. 更新速度和加速度
    !   5. 存储结果
  end subroutine
end module
```

### 3.2 显式动力分析（*DYNAMIC, EXPLICIT）

#### 3.2.1 基本理论

**中心差分法**：

```
速度更新：
v_{n+1/2} = v_{n-1/2} + (Δt_{n+1/2} + Δt_{n-1/2})/2 · a_n

位移更新：
u_{n+1} = u_n + Δt_{n+1/2} · v_{n+1/2}

加速度计算：
a_{n+1} = M^{-1} · (F_ext - F_int - C·v_{n+1/2})

稳定性条件：
Δt ≤ Δt_critical = 2/ω_max

其中：
- ω_max: 最大特征频率
```

#### 3.2.2 UFC实现

```fortran
! L5_RT 显式动力分析求解

    !    - 位移更新

    !    - 计算内力

    !    - 存储结果
  end subroutine
end module
```

### 3.3 模态分析（*FREQUENCY）

#### 3.3.1 基本理论

**广义特征值问题**：

```
K · φ = ω² · M · φ

其中：
- K: 刚度矩阵
- M: 质量矩阵
- φ: 特征向量（模态形状）
- ω: 特征频率（角频率）
```

**特征频率与周期**：

```
f = ω/(2π)  （频率，Hz）
T = 1/f = 2π/ω  （周期，s）
```

**模态质量归一化**：

```
φ^T · M · φ = 1
```

**UFC实现**：

```fortran

  public :: NM_SolveEigenvalue
  
  subroutine NM_SolveEigenvalue(K, M, eigenvalues, eigenvectors, n_modes)
    ! 求解广义特征值问题：K·φ = λ·M·φ
    ! 其中 λ = ω²
  
    ! 方法：Lanczos、子空间迭代、QR方法等
  end subroutine
end module
```

#### 3.3.2 频响分析（*FREQUENCY RESPONSE）

**频响函数**：

```
H(ω) = (-ω²M + iωC + K)^(-1)

其中：
- H(ω): 频响函数矩阵
- C: 阻尼矩阵
- i: 虚数单位
```

**响应计算**：

```
u(ω) = H(ω) · F(ω)

其中：
- F(ω): 频率域载荷
- u(ω): 频率域响应
```

**UFC实现位置**：`L5_RT/Dynamics/RT_FrequencyResponse.f90`

#### 3.3.3 屈曲分析（*BUCKLE）

**特征值屈曲问题**：

```
(K + λ·K_G) · φ = 0

其中：
- K: 线性刚度矩阵
- K_G: 几何刚度矩阵
- λ: 屈曲载荷因子
- φ: 屈曲模态
```

**临界载荷**：

```
P_critical = λ · P_ref

其中：
- P_ref: 参考载荷
- λ: 屈曲载荷因子
```

**UFC实现位置**：`L5_RT/Stability/RT_Buckling_Analysis.f90`

## 四、热分析

### 4.1 瞬态热传导（*HEAT TRANSFER）

#### 4.1.1 基本理论

**热传导方程**：

```
C · ∂T/∂t + K · T = Q

其中：
- C: 热容矩阵
- K: 热传导矩阵
- T: 温度向量
- Q: 热源向量
```

**θ-方法时间积分**：

```
C · (T_{n+1} - T_n)/Δt + K · [θ·T_{n+1} + (1-θ)·T_n] = Q_{n+θ}

其中：
- θ = 0: Forward Euler（显式，条件稳定）
- θ = 1/2: Crank-Nicolson（隐式，无条件稳定）
- θ = 1: Backward Euler（隐式，无条件稳定，最大数值阻尼）
```

#### 4.1.2 UFC实现

```fortran
! L5_RT 瞬态热传导求解
module RT_Solver_HeatTransfer
  public :: RT_SolveHeatTransfer
  
  subroutine RT_SolveHeatTransfer(model, step, results)
    type(MD_Model), intent(in) :: model
    type(StepDefinition), intent(in) :: step
    type(AnalysisResults), intent(out) :: results
  
    ! 瞬态热传导：
    ! 1. 组装热容矩阵C和热传导矩阵K
    ! 2. 时间积分循环：
    !    - 计算有效热传导矩阵：K_eff = C/Δt + θ·K
    !    - 计算有效热源：Q_eff = Q + C/Δt·T_n - (1-θ)·K·T_n
    !    - 求解：K_eff · T_{n+1} = Q_eff
    !    - 存储结果
  end subroutine
end module
```

### 4.2 稳态热传导（*HEAT TRANSFER, STEADY STATE）


## 五、耦合分析

### 5.1 热力耦合分析（*COUPLED TEMPERATURE-DISPLACEMENT）

#### 5.1.1 基本理论

**耦合方程**：

```
[M  0 ] [a]   [C_u  0  ] [v]   [K_uu  K_uT] [u]   [F_u]
[0  0 ] [Ṫ] + [0    C_T] [T] + [K_Tu  K_TT] [T] = [Q_T]

其中：
- M: 质量矩阵
- C_u: 结构阻尼矩阵
- C_T: 热容矩阵
- K_uu: 结构刚度矩阵
- K_uT: 热膨胀耦合矩阵
- K_Tu: 热-力耦合矩阵
- K_TT: 热传导矩阵
- u, v, a: 位移、速度、加速度向量
```

#### 6.4.2 收敛控制

**力收敛准则**：

```
|||R|| < ε_f · ||F_ext||

其中：
- R: 残差向量
- F_ext: 外载荷向量
- ε_f: 力收敛容差（默认：1.0e-3）
```

**位移收敛准则**：

```
|||Δu|| < ε_u · ||Δu_max||

其中：
- Δu: 位移增量
- Δu_max: 最大位移增量
- ε_u: 位移收敛容差（默认：1.0e-3）
```

**能量收敛准则**：

```
||Δu · R| < ε_e · |Δu_0 · R_0|

其中：
- Δu · R: 能量增量
- Δu_0 · R_0: 初始能量增量
- ε_e: 能量收敛容差（默认：1.0e-6）
```

**收敛检查策略**：

UFC采用多准则收敛检查，满足以下任一条件即认为收敛：

1. 力收敛 AND 位移收敛
2. 力收敛 AND 能量收敛
3. 位移收敛 AND 能量收敛

## 九、高级分析功能

### 9.1 自适应增量控制

**自适应增量策略**：

- **收敛良好**：增大增量步
- **收敛困难**：减小增量步
- **发散**：回退并减小增量步

**UFC实现**：

```fortran
! L5_RT 自适应增量控制
module RT_AdaptiveIncrement
  public :: RT_AdaptIncrement
  
  subroutine RT_AdaptIncrement(convergence_status, current_dt, new_dt)
    logical, intent(in) :: convergence_status
    real(wp), intent(in) :: current_dt
    real(wp), intent(out) :: new_dt
  
    if (convergence_status) then
      ! 收敛良好，增大增量步
      new_dt = current_dt * 1.2_wp
    else
      ! 收敛困难，减小增量步
      new_dt = current_dt / 2.0_wp
    end if
  end subroutine
end module
```

- 保存分析状态（位移、速度、加速度、应力、应变、状态变量）
- 从保存点恢复分析
- 支持增量步级别的重启

## 十、分析验证检查清单

### 10.1 静力学分析验证

- **平衡验证**：总外力 = 总反力
- **能量守恒**：总应变能 = 总外力功
- **收敛性**：Newton-Raphson迭代收敛
- **精度验证**：与理论解或ABAQUS结果对比

### 10.2 动力学分析验证

- **能量守恒**：总能量（动能+势能）守恒（无阻尼）
- **能量衰减**：总能量衰减符合预期（有阻尼）
- **频率精度**：固有频率与理论值一致
- **稳定性**：时间步长满足稳定性条件

### 10.3 热分析验证

- **能量守恒**：总热流 = 总热源
- **温度分布**：温度分布符合物理预期
- **稳态收敛**：稳态分析收敛到稳定解
