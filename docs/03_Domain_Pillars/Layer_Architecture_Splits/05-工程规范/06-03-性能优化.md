## 优化优先级

- P1(>30%): SpMV(稀疏矩阵), 本构更新
- P2(10-30%): 单元刚度, 直接求解器
- P3(<10%): IO, 初始化

## 性能目标

- Von Mises: 1.0µs/点
- SpMV(1M DOF): <100ms
- PCG 50步: <5秒
- 并行8核: >6倍加速
