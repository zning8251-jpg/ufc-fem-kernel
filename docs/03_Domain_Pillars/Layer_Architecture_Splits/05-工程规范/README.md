## 工程规范概述

### 编码规范

- 文件组织：MODULE/、TEST/、DOC/、CMAKE/
- 命名规范：前缀_主题_特征
- 缩进与注释：空格缩进，建议行长约 120 字符

### 测试策略

- L1 单元测试：pfUnit，目标高覆盖
- L2 集成测试：NAFEMS 等参考算例
- L3 系统测试：与参考软件对比验收

### 性能目标（示例）

- Von Mises：约 1.0µs/单元量级（视硬件与编译优化而定）
- SpMV（约 1M DOF）：毫秒级目标需结合具体实现评估
- 并行扩展性：随核数与问题规模调优
