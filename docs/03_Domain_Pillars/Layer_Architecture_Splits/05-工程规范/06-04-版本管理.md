## 版本规范

**版本格式**: MAJOR.MINOR.PATCH-PRERELEASE+BUILD
- Phase 2: 1.x.x
- Phase 3: 2.x.x
- 发布: v1.0.0

## 发布检查

- 30天前: 冻结API, L3单元测试, 文档
- 7天前: L2集成, Release Notes
- 1天前: L3系统, 性能检查

## 维护与缺陷

**缺陷等级**:
- P0 严重(<2h): 崩溃, 泄漏
- P1 重大(当天): 功能不可用
- P2 中等(一周): 功能不完整
- P3 轻微(下版): 文档、风格
