## 架构层差距

| 项目 | 现状 | 目标 |
| ------------ | -------------------------------- | ------------------------------ |
| 依赖方向 | 部分模块跨层引用 | 严格单向，无循环依赖 |
| Context 管理 | 线程局部 + 部分显式 | 全程显式 Context，Job 结束清理 |
| 模块聚合 | Material 已聚合，StepDriver 分散 | 核心域 50 个以内核心模块 |
| 接口统一 | 命名风格混杂 | UF_/MD_/RT_/AP_ 严格分层前缀 |

## 功能层差距

| 项目 | 现状 | 目标 |
| ------------ | ------------- | --------------------------- |
| 统一场论 | 分散 | UF_UnifiedField_Core 落地 |
| 多物理场耦合 | 有限 | 单块/交替/分区迭代三种策略 |
| 时间积分 | 隐式/显式分立 | 统一 θ/Newmark/广义α 框架 |
| 接触 | 聚合完成 | 通用接触、自适应、摩擦增强 |
| 单元库 | 分批迁移 | 壳/梁/实体/耦合/特殊全覆盖 |

## 求解层差距

| 项目 | 现状 | 目标 |
| -------- | -------------- | -------------------------- |
| 线性求解 | 直接/迭代 | 自动选择、预条件、GPU 加速 |
| 非线性 | NR/弧长/线搜索 | 收敛诊断、自适应策略 |
| 特征值 | 已有 | 模态、屈曲、子空间迭代 |
| 并行 | 多核 | MPI 分布式、负载均衡 |
| GPU | 部分内核 | 装配+求解全链路 GPU |

## 质量层差距

| 项目 | 现状 | 目标 |
| -------- | ---- | ------------------------ |
| 错误处理 | 分散 | 统一码、恢复、日志 |
| 测试 | 部分 | 单元+集成+回归 80% 覆盖 |
| 文档 | 分散 | 架构/API/用户/开发者体系 |
| 监控 | 部分 | 性能、内存、收敛诊断 |
