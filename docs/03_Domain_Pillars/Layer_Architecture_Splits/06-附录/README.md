## 附录概述

### 层内关键模块

- **L1_IF**：精度、常量、错误、内存管理
- **L2_NM**：线性/非线性求解、特征值、GPU 内核
- **L3_MD**：几何、材料、截面、载荷、Step 定义
- **L4_PH**：本构、单元、接触、时间积分
- **L5_RT**：Step 循环、装配、输出、Job 执行
- **L6_AP**：命令解析、Job 调度、用户 API

- 代码分析工具：子程序长度统计、文件大小统计、依赖审计
- 合并辅助工具：合并前分析、USE 语句更新
- 质量检查工具：代码质量检查
