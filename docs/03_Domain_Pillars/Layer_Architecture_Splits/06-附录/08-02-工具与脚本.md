## 代码分析工具

### 子程序长度统计

```bash
# 统计所有 .f90 文件中子程序的行数
python scripts/analyze_subroutine_length.py --path UFC/ufc_core
```

### 文件大小统计

```bash
# 统计文件大小，按行数排序
find UFC/ufc_core -name "*.f90" -exec wc -l {} + | sort -rn
```

### 依赖审计

```bash
# 生成依赖关系报告
python scripts/dep_audit.py --output DEPENDENCY_AUDIT_REPORT.md
```

## 合并辅助工具

### 合并前分析

```bash
# 分析待合并文件，预测合并后大小
python scripts/merge_analyzer.py --files file1.f90 file2.f90
```

### USE 语句更新

```bash
# 批量更新 USE 语句
python scripts/update_use_statements.py --old-module OLD_MOD --new-module NEW_MOD
```

## 质量检查工具

### 代码质量检查

```bash
# 检查文件大小、子程序长度、命名规范
python scripts/quality_check.py --path UFC/ufc_core
```
